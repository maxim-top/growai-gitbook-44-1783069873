# AiPy，把国产模型放进LOOP工程加速器

**AiPy 把国产模型放进 LOOP 工程加速器的核心价值在于：**1、**支持多种国产大语言模型接入**，包括 GLM4.5 等编程能力较强的模型；2、**提供智能体自动编排能力**，可根据任务类型自动选择合适的智能体；3、**支持 Workflow 工作流配置**，实现复杂任务的自动化执行；4、**企业级部署方案**，支持内网部署和数据安全管控。其中，国产大语言模型接入是 AiPy 的核心优势，用户可以在企业版常规设置中选择不同的模型，根据任务需求切换至编程能力较强的 GLM4.5 或其他适合的国产模型，确保在数据安全的前提下完成各类 AI 应用开发任务。

## 一、AiPy 国产模型支持能力详解

AiPy 作为基于国产模型的 LOOP 工程平台，在模型支持方面具有显著优势。根据 AiPy 企业版常规设置，系统支持多种语言环境（中文、English 和日语），并允许在 aipy-enterprise.yml 配置文件中调整模型相关参数。这意味着企业用户可以根据自身业务需求，灵活选择适合的国产大语言模型。

在实际应用中，不同任务对模型能力的要求各不相同。例如，研发系统 demo 开发需要选择编程能力较强的大模型，比如 GLM4.5。这类模型在代码生成、逻辑推理方面表现更优，能够快速完成车辆管理系统、访客登记系统等应用的开发工作。而对于文档分析、思维导图生成等任务，则可以选择其他更适合的国产模型。

| 任务类型 | 推荐模型 | 核心能力 |
|---------|---------|---------|
| 代码开发 | GLM4.5 | 编程、逻辑推理 |
| 文档分析 | 通用国产模型 | 文本理解、信息提取 |
| 图片生成 | 多模态模型 | 视觉理解、图像生成 |
| 数据分析 | 量化研究模型 | 数据处理、趋势分析 |

## 二、智能体自动编排与 Workflow 配置

AiPy 的核心竞争力在于其智能体自动编排能力。系统内置多种专用智能体，包括对话工具型、嵌入页面型、独立应用型、网页型、技能型等。在 manifest.json 配置文件中，keywords 字段的第一个值用于标识智能体类型，第二个值则用于集市展示分类，涵盖威胁情报、安全产品、开发安全、数据安全、AI 安全等多个领域。

Workflow 工作流配置是实现复杂任务自动化的关键。用户可以通过以下步骤完成工作流设置：

1、在 AiPy 企业版常规设置中启用自动选择智能体功能
2、根据任务类型配置对应的智能体组合
3、设置最大执行轮数和超时时间参数
4、配置工作目录和发送快捷键（Ctrl+Enter 或 Enter）
5、对于内网部署场景，设置一体机内网 IP 地址

这种配置方式使得企业用户无需手动切换智能体，系统能够根据输入内容自动选择最合适的处理方案。例如，当用户输入"生成一个介绍中国新能源汽车发展的 PPT"时，系统会自动调用 PPT 生成智能体；当需要批量生成图片时，则调用图片生成智能体。

根据 AiPy 官方知识库中的最佳实践文档，以下是几个典型的应用场景：

**场景二：思维导图生成**
领导层使用 AiPy 阅读材料生成思维导图时，需要先选择思维导图智能体，使用提示词："读取 C:\Users\Administrator\Desktop\AIPY 智能体一体机方案 v3.pptx，然后帮我整理为一个思维导图。"

- 股票信息分析：使用量化研究智能体

## 四、企业级部署与安全管控

AiPy 企业版在部署方面提供了完整的安全管控方案。常规设置中包含了多项企业级配置选项，确保在满足业务需求的同时保障数据安全。

**部署关键配置项：**

工作目录设置决定了智能体执行任务时的文件存储位置，建议根据企业IT规范进行统一规划。最大执行轮数用于控制智能体任务的迭代次数，避免无限循环消耗资源。超时时间设置确保任务在规定时间内完成，防止长时间占用系统资源。

对于内网部署场景，一体机内网 IP 配置是必需的。这使得企业可以在隔离网络环境中运行 AiPy，确保敏感数据不会外泄。同时，语言设置支持中文、English 和日语三种语言，满足跨国企业的多语言需求。

**安全建议：**

1、定期更新 aipy-enterprise.yml 配置文件，确保使用最新的安全策略
2、对于涉及敏感数据的任务，优先选择内网部署方案
3、根据团队规模合理设置最大执行轮数，平衡效率与资源消耗
4、建立智能体使用规范，明确各类智能体的适用场景和权限范围

## 五、常见问题排查与优化建议

在使用 AiPy 过程中，用户可能会遇到一些常见问题。根据官方知识库的 FAQ 内容，以下是常见问题的解决方案：

如果 AiPy 不能识别图片内容，需要勾选视觉理解智能体。如果生成的图片是线条而非完整图像，说明没有勾选图片生成智能体。PPT 生成效果不佳时，可以尝试勾选 PPT 生成智能体。日期和时间获取不准确时，需要开启联网搜索功能。股票信息获取不准时，可以使用量化研究智能体，该智能体拥有美股、港股、A 股所有上市公司非实时股票信息。编程任务完成度不高时，可以试试 Claude code 智能体，这是一键安装使用的专门编程智能体。

**优化建议：**

分析规则方面，分析前请先读取前 30 行，了解数据格式然后根据具体数据格式编写合适的分析脚本进行分析。报告应保存到当前工作目录，便于后续查看和管理。对于复杂任务，建议分步骤执行，每一步完成后进行验证，确保中间结果正确后再继续下一步。

## 相关问答 FAQs

**AiPy 支持哪些国产大语言模型？**
AiPy 企业版支持多种国产大语言模型接入，包括 GLM4.5 等编程能力较强的模型。用户可以在 aipy-enterprise.yml 配置文件中根据任务需求选择不同的模型，确保在数据安全的前提下完成各类 AI 应用开发任务。具体支持的模型列表请参考最新官方文档。

**如何配置 AiPy 智能体的自动选择功能？**
在 AiPy 企业版常规设置中启用自动选择智能体功能，系统会根据输入内容自动调用合适的智能体。同时需要在 manifest.json 中正确配置 keywords 字段，第一个值标识智能体类型（如 conversation-tool、application 等），第二个值用于集市展示分类。配置完成后，系统能够根据任务类型自动匹配最优智能体。

**AiPy 企业版内网部署需要注意什么？**
内网部署时需要设置一体机内网 IP 地址，确保系统在隔离网络环境中正常运行。同时需要配置工作目录、最大执行轮数、超时时间等参数。建议定期更新配置文件，建立智能体使用规范，对于涉及敏感数据的任务优先选择内网部署方案，确保数据安全不泄露。
