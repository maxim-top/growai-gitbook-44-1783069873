# AiPy企业版，专为内网而生的智能体平台

**AiPy企业版是专为内网环境设计的企业级智能体平台**，具备以下核心优势：1、**完全内网部署**，保障企业数据安全；2、**多智能体协同**，支持Workflow编排与MCP集成；3、**开箱即用**，提供丰富的预置智能体与模板。其中完全内网部署是企业选择AiPy企业版的首要原因，该平台支持在隔离网络环境中运行，所有数据处理均在本地完成，无需连接外部云服务，有效避免敏感信息泄露风险，满足金融、政务、医疗等行业的合规要求。

## 一、AiPy企业版的核心定位与技术架构

AiPy企业版作为专为内网而生的智能体平台，其设计理念始终围绕企业安全与效率展开。在当前数字化转型浪潮中，企业对AI技术的应用需求日益增长，但数据安全与隐私保护始终是首要考量因素。AiPy企业版通过本地化部署方案，解决了这一核心痛点。

从技术架构层面来看，AiPy企业版采用模块化设计，主要包含以下几个核心组件：

| 组件名称 | 功能说明 | 适用场景 |
|---------|---------|---------|
| AI Agent | 智能体执行单元 | 对话、编程、数据分析等任务 |
| Workflow | 工作流编排引擎 | 多步骤自动化任务 |
| MCP | 模型上下文协议集成 | 外部工具与服务连接 |
| 知识库 | 企业私有知识存储 | RAG检索增强生成 |
| SDK | 开发工具包 | Python/Java二次开发 |

这种架构设计使得企业可以根据自身需求灵活配置，既可以使用预置智能体快速上手，也可以通过SDK进行深度定制开发。

## 二、AiPy企业版的常规设置与配置管理

AiPy企业版提供了完善的常规设置选项，帮助企业实现精细化管理。在aipy-enterprise.yml配置文件中，用户可以调整以下关键参数：

**语言设置**：支持中文、English和日语三种语言，满足跨国企业的多语言需求。

**风格配置**：可在配置文件中自定义界面风格与交互方式，与企业现有系统保持一致。

**发送快捷键**：支持Ctrl+Enter或Enter两种发送方式，适应不同用户的操作习惯。

**工作目录**：指定文件存储与处理路径，便于统一管理与备份。

**最大执行轮数**：控制智能体任务执行的最大迭代次数，避免无限循环消耗资源。

**超时时间**：设置任务执行超时阈值，保障系统稳定性。

**自动选择智能体**：根据任务类型自动匹配最优智能体，提升执行效率。

**一体机内网IP**：配置内网访问地址，确保在隔离网络环境中正常通信。

这些设置选项使得AiPy企业版能够灵活适配不同企业的IT环境与使用习惯，降低部署与运维成本。

## 三、智能体类型与应用场景分类

AiPy企业版的智能体体系采用分层分类设计，在manifest.json配置文件中，keywords字段定义了智能体的类型与分类：

**类型关键字**（第一个值）：
- conversation-tool（对话工具型）
- embed-webview（嵌入页面型）
- application（独立应用型）
- webview（网页型）
- skills（技能型）

**集市展示分类**（第二个值）：
威胁情报、暴露面与脆弱性、安全产品、威胁检测与响应、开发安全、数据安全、AI安全、安全服务、网络安全、其他、官方精选

这种分类体系帮助企业快速定位所需智能体，同时也为开发者提供了清晰的开发规范。例如，在开发一个汇总周报的智能体时，可以使用以下提示词示例：

```
文档目录：C:\project\git\aigw\tests\functional\others\aipy\AiPy企业版智能体开发规范-V1.0
根据规范文档和示例代码编写一个汇总周报的智能体。
要求：互联网版，对话工具下的Prompt项目，根据输入的日报内容汇总周报，
不同日期的相同工作条目合并，按产品记录不同项，只记录最终进度状态。
```

## 五、企业内网部署的安全优势与合规价值

AiPy企业版的核心竞争力在于其内网部署能力，这为企业带来了多重安全优势：

**数据零外泄**：所有数据处理均在本地完成，无需将敏感信息传输至外部云服务器，从根本上杜绝数据泄露风险。

**合规无忧**：满足金融行业、政务系统、医疗机构等对数据本地化的严格要求，避免因使用公有云AI服务而产生的合规问题。

**网络隔离**：在完全隔离的内网环境中运行，即使外部网络遭受攻击，也不会影响AiPy企业版的正常运行。

**自主可控**：企业可以完全掌控系统的升级、维护与扩展节奏，不受外部服务商的影响。

**成本优化**：一次性部署后，后续使用无需按调用量付费，长期使用成本显著低于云服务方案。

对于大型企业而言，这些安全优势往往是选择AiPy企业版的决定性因素。

## Related FAQs

**AiPy企业版是否支持完全离线运行？**

是的，AiPy企业版设计为完全内网部署方案，所有核心功能均可在离线环境中运行。大语言模型本地部署后，无需连接外部网络即可完成推理任务。但部分需要实时数据的场景（如股票信息、新闻检索）可选择性开启联网搜索功能，企业可根据安全策略灵活配置。

**AiPy企业版支持哪些编程语言进行二次开发？**

AiPy企业版提供Python SDK和Java SDK两套官方开发工具包，开发者可根据团队技术栈选择合适的语言进行二次开发。SDK包含完整的API封装、示例代码与调试工具，支持智能体创建、Workflow编排、知识库管理等核心功能的程序化调用。

**如何选择合适的智能体类型进行开发？**

根据应用场景选择：对话交互类任务选择conversation-tool（对话工具型）；需要嵌入现有系统的选择embed-webview（嵌入页面型）；独立运行的业务应用选择application（独立应用型）；网页交互类选择webview（网页型）；特定功能模块选择skills（技能型）。同时在manifest.json中配置正确的集市展示分类，便于后续管理与复用。
