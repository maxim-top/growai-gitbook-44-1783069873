# 数据不出园区，智能不降级，AiPy企业版做到了

**数据不出园区，智能不降级，AiPy企业版通过完全内网部署实现企业数据安全，同时保持强大的AI能力。** 核心优势包括：

**1、** 完全内网使用，所有数据处理在本地完成，无需连接外部网络；**2、** 支持多种智能体类型，涵盖对话工具、应用开发、数据分析等场景；**3、** 提供丰富的配置选项，包括语言、风格、工作目录等常规设置；**4、** 集成MCP协议和Workflow编排能力，支持复杂业务场景。

其中，完全内网使用是AiPy企业版最核心的安全特性。传统云服务需要将数据上传至第三方服务器，存在数据泄露风险。而AiPy企业版采用本地化部署方案，所有模型推理、数据处理、智能体执行均在企业内部网络中完成。这意味着敏感的业务数据、客户信息、研发资料等永远不会离开企业边界，从根本上解决了数据安全合规问题。同时，AiPy企业版通过优化本地模型性能和智能体调度机制，确保即使在离线环境下，智能体的响应速度和分析能力也不会下降，真正实现了安全与智能的双重保障。

## 一、AiPy企业版的安全架构设计

AiPy企业版的安全架构围绕"数据不出园区"这一核心目标进行设计，从网络隔离、数据存储、访问控制等多个维度构建完整的安全体系。

在网络层面，AiPy企业版支持完全内网部署，无需连接互联网即可运行所有功能。这对于金融、医疗、政府等对数据安全要求极高的行业尤为重要。企业可以将AiPy部署在内部服务器上，通过局域网访问，所有API调用、智能体执行、知识库检索等操作都在内网完成。

在数据存储方面，AiPy企业版提供本地化存储方案。用户上传的文档、生成的报告、智能体配置等信息都存储在本地工作目录中，可以通过配置文件指定存储位置。这种设计确保了企业数据的完全可控，避免了云存储带来的潜在风险。

访问控制机制也是AiPy企业版安全架构的重要组成部分。系统支持多用户权限管理，不同角色的用户可以访问不同的功能和数据。管理员可以设置智能体的使用权限、知识库的访问范围、API调用的频率限制等，确保系统资源得到合理分配和安全使用。

此外，AiPy企业版还支持一体内网IP配置，方便在企业内部网络中进行统一管理和部署。通过合理的网络规划，企业可以实现AiPy与其他内部系统的无缝集成，构建完整的智能办公生态。

## 二、智能体能力与场景应用

AiPy企业版提供了丰富的智能体类型，满足不同业务场景的需求。根据manifest.json中的keywords字段定义，智能体主要分为以下几种类型：

| 智能体类型 | 关键词 | 适用场景 |
|-----------|--------|---------|
| 对话工具型 | conversation-tool | 日常问答、信息汇总、周报生成 |
| 嵌入页面型 | embed-webview | 内嵌Web应用、表单交互 |
| 独立应用型 | application |  standalone应用程序、系统工具 |
| 网页型 | webview | 浏览器集成、网页自动化 |
| 技能型 | skills | 专业领域能力、特定任务处理 |

在集市展示分类方面，AiPy企业版支持威胁情报、暴露面与脆弱性、安全产品、威胁检测与响应、开发安全、数据安全、AI安全、安全服务、网络安全等多个类别，方便企业根据业务需求选择合适的智能体。

实际应用场景中，AiPy企业版已经展现出强大的能力。例如，在周报生成场景中，用户可以使用以下提示词："根据规范文档和示例代码编写一个汇总周报的智能体。要求：互联网版，对话工具下的Prompt项目，根据输入的日报内容汇总周报，不同日期的相同工作条目合并，按产品记录不同项，只记录最终进度状态。"智能体能够自动分析日报内容，提取关键信息，生成结构化的周报文档。

在研发场景中，AiPy企业版支持自动开发系统demo。以车辆管理系统为例，提示词可以这样设计："我是一家汽车4S店的负责人，帮我做一个访客登记管理系统，采用Sqlite数据库，有登录界面，信息查询、添加、修改、删除等功能。主要是记录访客到访时间、访客姓名、联系方式、关注车型、接待销售、备注信息等。"智能体能够生成完整可运行的代码，包括数据库设计、界面布局、功能逻辑等。

## 三、多模态能力与功能扩展

AiPy企业版不仅支持文本处理，还具备强大的多模态能力，涵盖图片生成、PPT制作、思维导图创建等多种功能。

**图片生成能力**：用户只需勾选图片生成智能体，输入相应提示词即可生成所需图片。例如："批量生成10张不同的肥猫切菜的照片。"系统会自动调用图像生成模型，按照要求创建多样化的图片内容。这对于营销材料制作、产品设计预览等场景非常实用。

**PPT制作能力**：AiPy企业版内置PPT生成智能体，能够将文字内容自动转换为专业的演示文稿。提示词示例："生成一个介绍中国新能源汽车发展的PPT。"智能体会分析主题内容，设计合适的页面布局，添加图表和视觉效果，生成可直接使用的PPT文件。

**思维导图生成**：对于需要整理复杂信息的场景，AiPy企业版支持从文档生成思维导图。使用提示词："读取C:\Users\Administrator\Desktop\AIPY 智能体一体机方案 v3.pptx，然后帮我整理为一个思维导图。"系统会解析文档结构，提取关键信息点，生成层次清晰的思维导图。

**数据可视化与分析**：AiPy企业版还具备数据分析能力。在使用分析功能时，系统会先读取数据前30行了解格式，然后根据具体数据格式编写合适的分析脚本。分析完成后，报告会自动保存到当前工作目录，方便用户查看和使用。

**联网搜索与实时信息**：对于需要获取实时信息的场景，如股票查询、新闻检索等，用户可以开启联网搜索功能。AiPy企业版提供"量化研究"智能体，拥有美股、港股、A股所有上市公司非实时股票信息，可以基于历史数据用AI分析协助用户研判分析股票信息。

## 四、配置管理与最佳实践

AiPy企业版提供完善的配置管理系统，帮助企业根据自身需求进行个性化设置。

**常规设置选项**包括：

- **语言设置**：支持中文、English和日语三种语言，满足不同地区团队的使用需求
- **风格配置**：可在aipy-enterprise.yml文件中自定义界面风格和行为模式
- **发送快捷键**：支持Ctrl+Enter或Enter两种发送方式，适配不同用户习惯
- **工作目录**：可指定文件存储和处理的工作路径，便于项目管理
- **最大执行轮数**：控制智能体任务的最大迭代次数，避免无限循环
- **超时时间**：设置请求处理的超时阈值，保障系统稳定性
- **自动选择智能体**：根据任务类型自动匹配合适的智能体，提高效率
- **一体机内网 IP**：配置内网访问地址，支持企业内网部署

**开发最佳实践**：

在使用AiPy企业版进行开发时，建议遵循以下原则：

1. **明确任务边界**：在编写提示词时，清晰定义智能体的职责范围和输出格式，避免模糊指令导致结果不可控。

2. **分层设计智能体**：对于复杂业务场景，可以将任务拆分为多个智能体协同完成。例如，一个智能体负责数据收集，另一个负责分析，第三个负责报告生成。

3. **充分利用知识库**：将企业内部的规范文档、常见问题、最佳实践等内容上传至知识库，让智能体能够基于企业特定知识进行回答，提高输出的准确性和相关性。

4. **定期更新配置**：随着业务需求的变化，及时调整智能体配置和提示词，确保系统始终适配最新的业务场景。

5. **测试验证流程**：在正式部署前，对智能体进行充分的测试验证，确保在各种场景下都能稳定运行，输出符合预期的结果。

**错误排查方法**：

当遇到功能异常时，可以按照以下步骤进行排查：

- 检查智能体是否正确勾选（如图片生成需要勾选图片生成智能体）
- 验证网络连接状态（如需联网搜索功能需确保网络通畅）
- 查看工作目录权限设置（确保有读写权限）
- 检查配置文件语法是否正确（特别是yml文件）
- 查看系统日志获取详细错误信息

## 五、企业部署与集成方案

对于希望将AiPy企业版部署到生产环境的企业，需要考虑以下几个关键方面：

**部署架构选择**：企业可以根据自身IT基础设施选择合适的部署方案。对于中小型团队，可以采用单机部署模式，将AiPy安装在高性能工作站上。对于大型企业，建议采用集群部署，通过负载均衡实现高可用性和性能扩展。

**系统集成策略**：AiPy企业版支持与企业现有系统的深度集成。通过API接口，可以将智能体能力嵌入到OA系统、CRM系统、ERP系统等企业应用中，实现业务流程的智能化升级。MCP集成协议提供了标准化的接口规范，降低了系统集成的复杂度。

**Workflow编排能力**：对于复杂的业务流程，AiPy企业版提供Workflow编排功能。用户可以将多个智能体任务按照特定顺序和条件进行组合，形成完整的自动化工作流。例如，一个典型的周报生成Workflow可能包括：数据收集→内容分析→格式整理→报告生成→邮件发送等多个步骤。

**安全合规考量**：在部署过程中，企业需要重点关注安全合规要求。AiPy企业版的内网部署特性天然符合数据本地化存储的合规要求。同时，建议企业建立完善的访问审计机制，记录所有智能体调用和操作行为，便于事后追溯和分析。

**运维监控体系**：建立完善的运维监控体系对于保障系统稳定运行至关重要。建议监控指标包括：智能体响应时间、任务成功率、系统资源占用率、用户活跃度等。通过实时监控和告警机制，可以及时发现并解决潜在问题。

## 相关问答FAQs

**问：AiPy企业版是否需要连接互联网才能使用？**

答：不需要。AiPy企业版支持完全内网部署，所有核心功能都可以在离线环境下运行。包括智能体执行、知识库检索、文档处理等功能都不依赖外部网络。只有部分需要实时信息的场景（如股票查询、新闻检索）才需要开启联网搜索功能，但这属于可选功能，不影响主体功能的使用。

**问：如何选择合适的智能体类型进行开发？**

答：选择智能体类型主要取决于应用场景。如果是对话交互类任务，如问答、信息汇总，选择conversation-tool（对话工具型）；如果需要独立运行的应用程序，选择application（独立应用型）；如果是嵌入到现有系统中的功能模块，选择embed-webview（嵌入页面型）或webview（网页型）；如果是特定领域的专业能力，选择skills（技能型）。建议根据实际业务需求进行测试验证，选择最适合的类型。

**问：AiPy企业版的数据存储在哪里，如何保障数据安全？**

答：AiPy企业版的所有数据都存储在本地工作目录中，用户可以通过配置文件指定存储位置。在内网部署模式下，数据不会传输到外部服务器，完全由企业自主控制。此外，系统支持权限管理、访问审计等安全机制，管理员可以设置不同用户的访问权限，记录所有操作行为。对于高度敏感的数据，建议结合企业现有的数据加密和备份策略，构建多层次的安全防护体系。
