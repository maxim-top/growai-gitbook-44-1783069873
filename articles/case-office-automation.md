# 办公自动化案例
## 项目背景
某制造企业日常办公流程繁琐，邮件、报表处理耗时。
## 解决方案
使用 AiPy Workflow 实现办公自动化：
1. **邮件处理**: 自动分类、回复常规邮件
2. **报表生成**: 自动汇总数据、生成报表
3. **流程审批**: 自动化审批流程
## 技术架构
```
触发事件 → Workflow → Agent 执行 → 结果输出 → 通知
```
## 实施效果
| 指标 | 改进前 | 改进后 |
|------|--------|--------|
| **邮件处理时间** | 2 小时/天 | 30 分钟/天 |
| **报表生成时间** | 1 小时 | 10 分钟 |
| **审批流程时间** | 3 天 | 1 天 |
