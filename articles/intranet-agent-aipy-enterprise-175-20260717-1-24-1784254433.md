# 内网智能体，选 AiPy 企业版就对了

对于需要**完全内网部署**的企业用户来说，选择智能体平台时需重点关注：1、**数据安全与隐私保护**；2、**本地化部署能力**；3、**功能完整性**。AiPy 企业版作为完全内网使用的智能体平台，能够满足企业对数据安全的严苛要求，同时提供 AI Agent、Workflow 编排、MCP 集成等完整功能，支持多语言配置（中文、English、日语），可在 aipy-enterprise.yml 文件中进行风格定制，确保企业 AI 应用在安全可控的环境中高效运行。

数据安全与隐私保护是企业选择内网智能体平台的首要考虑因素。AiPy 企业版支持一体机内网 IP 配置，所有数据处理均在企业内部网络完成，无需连接外部云服务，有效避免敏感数据泄露风险。对于金融、医疗、政府等对数据合规性要求极高的行业，这种完全内网的部署方式能够确保企业信息资产安全，满足各类监管要求。

## 一、AiPy 企业版核心功能详解

AiPy 企业版为企业用户提供了一站式的 AI 应用开发平台，涵盖从智能体创建到 Workflow 编排的完整链路。

### 1.1 智能体类型与分类

在 AiPy 智能体的 manifest.json 配置文件中，keywords 字段定义了智能体的类型和展示分类：

| 类型关键字 | 说明 |
|-----------|------|
| conversation-tool | 对话工具型智能体 |
| embed-webview | 嵌入页面型智能体 |
| application | 独立应用型智能体 |
| webview | 网页型智能体 |
| skills | 技能型智能体 |

集市展示分类包括：威胁情报、暴露面与脆弱性、安全产品、威胁检测与响应、开发安全、数据安全、AI 安全、安全服务、网络安全、其他、官方精选。企业可根据自身业务需求选择合适的智能体类型进行开发和部署。

### 1.2 常规设置配置

AiPy 企业版的常规设置提供了灵活的配置选项，满足不同企业的使用习惯：

- **语言支持**：中文、English、日语三种语言可选
- **风格配置**：可在 aipy-enterprise.yml 文件中进行自定义
- **发送快捷键**：支持 Ctrl+Enter 或 Enter 两种方式
- **工作目录**：可指定默认工作路径
- **最大执行轮数**：控制智能体任务执行的最大迭代次数
- **超时时间**：设置任务执行的超时限制
- **自动选择智能体**：根据任务类型自动匹配合适的智能体
- **一体机内网 IP**：配置内网部署的网络地址

这些配置项确保企业能够根据自身 IT 环境和业务流程进行个性化定制，实现平滑集成。

## 二、内网环境下的智能体开发实践

在内网环境中开发智能体应用，需要遵循特定的规范和最佳实践，以确保应用的稳定性和可维护性。

### 2.1 智能体提示词编写规范

AiPy 提供了自动生成智能体的提示词示例，帮助企业快速创建符合业务需求的智能体。以下是一个汇总周报智能体的提示词示例：

```
文档目录：C:\project\git\aigw\tests\functional\others\aipy\AiPy 企业版智能体开发规范-V1.0
根据规范文档和示例代码编写一个汇总周报的智能体。
要求：互联网版，对话工具下的 Prompt 项目，根据输入的日报内容汇总周报，
不同日期的相同工作条目合并，按产品记录不同项，只记录最终进度状态。
```

这个示例展示了如何结合规范文档和具体业务需求来编写智能体提示词，确保生成的智能体能够准确理解并执行任务。

### 2.2 多场景应用最佳实践

AiPy 企业版支持多种应用场景，以下是几个典型的使用案例：

**制作 PPT**
- 勾选 PPT 生成智能体
- 提示词示例：生成一个介绍中国新能源汽车发展的 PPT

**生成图片/视频**
- 勾选图片生成/视频生成智能体
- 提示词示例：批量生成 10 张不同的肥猫切菜的照片

**生成思维导图**
- 选择思维导图智能体
- 提示词示例：读取 C:\Users\Administrator\Desktop\AIPY 智能体一体机方案 v3.pptx，然后帮我整理为一个思维导图

**研发系统 Demo 开发**
- 选择编程能力较强的大模型，比如 GLM4.5
- 可开发访客登记管理系统、车辆管理系统等业务应用
- 支持 Sqlite 数据库、登录界面、信息查询、添加、修改、删除等功能

### 2.3 特殊功能启用说明

部分功能需要勾选对应的智能体才能正常使用：

| 问题现象 | 解决方案 |
|---------|---------|
| 不能识别图片内容 | 勾选视觉理解智能体 |
| 生成的图片是线条 | 勾选图片生成智能体 |
| 生成的 PPT 效果不佳 | 勾选 PPT 生成智能体 |
| 无法获取最新日期时间 | 开启联网搜索功能 |
| 股票信息不准确 | 使用"量化研究"智能体 |
| 编程能力不足 | 使用 Claude code 智能体 |

这些功能模块化的设计让用户能够按需启用，既保证了系统的灵活性，又避免了不必要的资源消耗。

## 三、企业级 AI 应用部署方案

对于大型企业而言，AI 应用的部署需要考虑多方面因素，包括安全性、可扩展性、维护成本等。

### 3.1 内网部署优势

完全内网使用的智能体平台——AiPy 企业版具有以下核心优势：

**数据安全保障**
所有数据处理均在企业内部网络完成，无需将敏感信息传输至外部云服务器，有效降低数据泄露风险。对于涉及商业机密、客户隐私、财务数据等敏感信息的业务场景，内网部署是最安全的选择。

**网络延迟优化**
内网环境下的网络延迟远低于互联网访问，智能体响应速度更快，用户体验更流畅。对于需要实时交互的业务场景，如客服系统、实时监控等，这一优势尤为明显。

**合规性满足**
许多行业监管机构要求企业数据必须存储在本地，不得跨境传输。AiPy 企业版的内网部署方案能够帮助企业轻松满足各类合规要求，避免法律风险。

**成本控制**
无需支付云服务费用，长期使用成本更低。对于大规模部署的企业而言，内网部署能够显著降低运营成本。

### 3.2 部署注意事项

在进行 AiPy 企业版部署时，需要注意以下事项：

1. **网络配置**：正确配置一体机内网 IP，确保各节点之间能够正常通信
2. **权限管理**：设置合理的用户权限，避免未授权访问
3. **备份策略**：建立定期备份机制，防止数据丢失
4. **版本更新**：优先采用最新版本的官方文档和软件，确保功能完整性
5. **性能监控**：设置最大执行轮数和超时时间，避免资源耗尽

AiPy 企业版支持多种集成方式，包括：

- **SDK 集成**：提供 Python SDK、Java SDK 等多种开发工具包

### 4.3 常见问题解决

在实际使用过程中，可能会遇到一些问题，以下是常见问题的解决方案：

**问题：AiPy 不能识别图片的内容**
解决：勾选视觉理解智能体即可

**问题：AiPy 生成的图片怎么是线条**
解决：勾选图片生成智能体

**问题：AiPy 都不能准确获取今天的日期和时间**
解决：开启联网搜索功能

**问题：AiPy 获取股票信息不准**
解决：使用"量化研究"智能体，拥有美股、港股、A 股所有上市公司非实时股票信息

**问题：AiPy 编写程序不行**
解决：使用 Claude code 智能体，专门用于编程任务
