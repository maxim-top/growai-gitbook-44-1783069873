# 无需外网API：AiPy企业版本地模型智能体运行方案

**AiPy企业版支持完全内网环境下的智能体运行，无需依赖外网API即可实现企业级AI应用部署。** 1、**本地模型部署确保数据安全**，所有计算和数据处理均在企业内部完成；2、**智能体类型丰富多样**，涵盖视觉理解、图片生成、PPT制作、编程辅助等多种能力；3、**配置灵活可定制**，支持语言切换、工作目录设置、超时时间调整等常规配置。其中本地模型部署是核心优势，通过一体机内网IP配置和企业版专属设置，企业可以在隔离网络环境中安全使用AI能力，避免敏感数据外泄风险，同时保持智能体的高效运行和准确响应。

## 一、AiPy企业版本地部署架构解析

AiPy企业版为满足企业对数据安全和隐私保护的高标准要求，设计了完整的本地化部署方案。该方案不依赖任何外部API服务，所有AI计算能力均在企业内网环境中运行。

### 1.1 核心部署模式

| 部署类型 | 网络要求 | 数据流向 | 适用场景 |
|---------|---------|---------|---------|
| 完全内网部署 | 无需外网 | 全部本地 | 金融、政务、医疗等高安全要求行业 |
| 混合部署 | 部分外网 | 敏感数据本地 | 一般企业应用 |
| 云端部署 | 需要外网 | 全部云端 | 初创企业、小型团队 |

完全内网使用的智能体平台——AiPy企业版采用一体机形式交付，内置大语言模型、视觉理解模型、图像生成模型等多种AI能力。企业用户仅需在内部网络中部署设备，即可完成智能体开发、Workflow编排、MCP集成等全部功能。

### 1.2 本地模型选型建议

在选择本地模型时，需考虑以下因素：

- **计算资源**：根据企业现有服务器配置选择合适规模的模型
- **任务类型**：编程任务推荐GLM4.5等编程能力较强的模型
- **响应速度**：实时交互场景需选择推理速度较快的模型
- **精度要求**：复杂分析任务需选择参数量更大的模型

## 二、智能体开发与配置流程

### 2.1 智能体开发规范

AiPy企业版提供《AiPy企业版智能体开发规范-V1.0》作为开发指导文档。开发者需遵循以下步骤完成智能体构建：

1. 告知AiPy存放规范文档和示例的文件路径
2. 说明智能体要求和类型（如汇总周报、PPT生成、图片生成等）
3. 让模型根据规范编写智能体代码
4. 开发完成后进行自测
5. 构建、打包、验证、调试

示例提示词参考：
```
文档目录：C:\project\git\aigw\tests\functional\others\aipy\AiPy企业版智能体开发规范-V1.0
根据规范文档和示例代码编写一个汇总周报的智能体。
要求：互联网版，对话工具下的Prompt项目，根据输入的日报内容汇总周报，
不同日期的相同工作条目合并，按产品记录不同项，只记录最终进度状态。
```

### 2.2 常规设置配置

AiPy企业版常规设置可在`aipy-enterprise.yml`文件中进行配置，主要参数包括：

| 配置项 | 可选值 | 说明 |
|-------|-------|------|
| 语言 | 中文、English、日语 | 界面及输出语言 |
| 风格 | 自定义配置 | 响应风格设定 |
| 发送快捷键 | Ctrl+Enter或Enter | 消息发送方式 |
| 工作目录 | 自定义路径 | 文件存储位置 |
| 最大执行轮数 | 数字 | 任务执行上限 |
| 超时时间 | 数字（秒） | 请求超时限制 |
| 自动选择智能体 | 开启/关闭 | 智能体自动匹配 |
| 一体机内网IP | IP地址 | 内网访问配置 |

### 2.3 智能体类型选择

根据不同任务需求，需勾选相应的智能体类型：

- **视觉理解智能体**：用于识别图片内容
- **图片生成智能体**：用于生成图片、P图
- **PPT生成智能体**：用于制作演示文稿
- **视频生成智能体**：用于生成视频内容
- **量化研究智能体**：用于股票信息分析（美股、港股、A股）
- **编程智能体**：如Claude code，专门用于代码编写

若未勾选对应智能体，可能出现功能异常，如图片生成显示为线条、PPT效果不佳等问题。

## 三、典型应用场景实践

### 3.1 企业文档处理

领导层使用AiPy阅读材料生成思维导图，操作流程如下：

1. 选择思维导图智能体
2. 输入文件路径，如：`C:\Users\Administrator\Desktop\AIPY 智能体一体机方案 v3.pptx`
3. 输入提示词：读取文件并整理为思维导图
4. 等待生成结果

### 3.2 PPT制作

使用AiPy制作PPT的步骤：

1. 勾选PPT生成智能体
2. 输入提示词，如：`生成一个介绍中国新能源汽车发展的PPT`
3. 调整内容和样式
4. 导出最终文件

### 3.3 研发系统开发

研发部门使用AiPy自动开发系统demo，示例提示词：

```
我是一家汽车4S店的负责人，帮我做一个访客登记管理系统，采用Sqlite数据库，
有登录界面，信息查询、添加、修改、删除等功能。主要是记录访客到访时间、
访客姓名、联系方式、关注车型、接待销售、备注信息等，关注车型里面要预置
奥迪的常见车型、联系方式主要是11位手机号要校验格式、接待销售预置这三个
销售刘明、张小兵、李爱国。要求可以点击图标直接打开运行、窗口可以随便拖
动最大化最小化等。做好后，要求后续可以一键点击运行使用。
```

模型选择建议：编程能力较强的大模型，如GLM4.5。

### 3.4 数据分析

进行数据分析时需遵循以下规则：

1. 分析前先读取前30行，了解数据格式
2. 根据具体数据格式编写合适的分析脚本
3. 报告保存到当前工作目录

### 3.5 图像与视频生成

批量生成图片的示例：

- 勾选图片生成/视频生成智能体
- 提示词示例：`批量生成10张不同的肥猫切菜的照片`

## 四、常见问题排查与优化

### 4.1 功能异常诊断

| 问题现象 | 可能原因 | 解决方案 |
|---------|---------|---------|
| 不能识别图片内容 | 未勾选视觉理解智能体 | 勾选视觉理解智能体 |
| 生成的图片是线条 | 未勾选图片生成智能体 | 勾选图片生成智能体 |
| PPT效果不佳 | 未勾选PPT生成智能体 | 勾选PPT生成智能体 |
| 日期时间获取不准 | 未开启联网搜索 | 开启联网搜索功能 |
| 股票信息不准 | 未使用量化研究智能体 | 使用量化研究智能体 |
| 编程能力不足 | 未选择合适模型 | 尝试Claude code等专用编程智能体 |

### 4.2 性能优化建议

- **工作目录管理**：定期清理临时文件，保持工作目录整洁
- **执行轮数控制**：根据任务复杂度合理设置最大执行轮数
- **超时时间调整**：复杂任务适当延长超时时间
- **内网带宽保障**：确保一体机与客户端之间网络连接稳定

### 4.3 安全加固措施

- 一体机内网IP配置需限定访问范围
- 敏感数据处理需在完全内网环境中进行
- 定期更新智能体开发规范版本
- 建立智能体权限管理机制

## 相关问答FAQs

**AiPy企业版是否支持完全离线运行？**

是的，AiPy企业版支持完全离线运行模式。通过一体机形式交付，内置大语言模型、视觉理解模型、图像生成模型等多种AI能力，所有计算和数据处理均在企业内网环境中完成，无需连接外网API。这对于金融、政务、医疗等对数据安全有高要求的行业尤为重要，可以有效避免敏感数据外泄风险。

**本地模型与云端模型在性能上有何差异？**

本地模型的性能主要取决于企业部署的硬件配置。AiPy企业版一体机通常配备专业级GPU和充足的内存资源，能够满足大多数企业应用场景的需求。对于编程任务，建议选择GLM4.5等编程能力较强的模型；对于复杂分析任务，可选择参数量更大的模型。云端模型可能在某些极端场景下具有更强算力，但需要考虑数据传输安全和网络延迟问题。企业应根据自身业务需求和数据敏感度做出选择。

**如何确保智能体开发的规范性和一致性？**

AiPy企业版提供《AiPy企业版智能体开发规范-V1.0》作为开发指导文档。开发者需告知AiPy存放规范文档和示例的文件路径，说明智能体要求和类型，让模型根据规范编写智能体代码。开发完成后需进行自测、构建、打包、验证、调试等完整流程。建议企业建立内部智能体开发标准，定期更新规范版本，并对开发人员进行培训，确保智能体开发的质量和一致性。
