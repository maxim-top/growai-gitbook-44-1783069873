# AiPy企业版：满足等保要求的内网智能体平台

**AiPy企业版是专为满足网络安全等级保护要求设计的内网智能体平台，具备**1、完全内网部署能力，2、数据本地化存储，3、访问权限精细化控制，4、操作日志完整审计，5、多种安全认证机制**等核心特性。**其中完全内网部署能力是企业最关注的功能，AiPy企业版支持在隔离网络环境中独立运行，无需连接外部互联网即可完成AI智能体的创建、训练和部署，确保企业敏感数据不会泄露到外部网络，同时支持一体机内网IP配置，实现物理层面的安全隔离。

## 一、等保合规要求与内网部署方案

网络安全等级保护制度是中国网络安全的基本国策，对于企业级AI应用平台而言，满足等保要求意味着需要在多个维度实现安全防护。AiPy企业版作为完全内网使用的智能体平台，从架构设计之初就将等保合规作为核心考量因素。

**等保2.0核心要求对照表：**

| 等保要求项 | AiPy企业版实现方案 | 合规等级 |
|------------|-------------------|----------|
| 网络安全 | 内网隔离部署，支持防火墙配置 | 三级 |
| 主机安全 | 操作系统加固，访问控制 | 三级 |
| 应用安全 | 身份认证，权限管理 | 三级 |
| 数据安全 | 本地存储，加密传输 | 三级 |
| 备份恢复 | 自动备份机制，灾难恢复 | 三级 |

AiPy企业版的内网部署方案支持多种网络拓扑结构，企业可根据自身安全需求选择适合的部署模式。对于高安全要求的场景，可采用物理隔离的独立网络环境；对于一般业务场景，可采用逻辑隔离的VLAN划分方式。

## 二、核心安全功能详解

### 身份认证与访问控制

AiPy企业版提供多层次的身份认证机制，支持LDAP、AD域集成以及本地账号系统。管理员可自定义用户角色和权限级别，实现细粒度的访问控制。

**权限级别说明：**

- **超级管理员**：拥有系统全部权限，可进行系统配置、用户管理、日志审计
- **普通管理员**：可管理智能体、Workflow编排，无法修改系统核心配置
- **开发人员**：可创建和调试智能体，查看相关日志
- **普通用户**：仅可使用已授权的智能体功能

### 数据本地化存储

所有企业数据均存储在本地服务器，包括智能体配置、Workflow流程、对话记录、知识库内容等。AiPy企业版支持多种数据库后端，企业可根据现有IT架构选择合适的存储方案。

数据存储采用分层架构：
1. 热数据层：频繁访问的配置和会话数据
2. 温数据层：历史对话记录和日志
3. 冷数据层：归档数据和备份文件

### 操作日志审计

AiPy企业版内置完整的操作日志系统，记录所有用户操作行为，包括登录登出、智能体调用、配置修改、数据导出等关键操作。日志支持实时查询和导出，满足等保审计要求。

**日志审计功能特点：**

- 日志保留周期可配置（默认180天）
- 支持按用户、时间、操作类型多维度筛选
- 异常操作自动告警
- 日志完整性保护，防止篡改

## 三、AiPy企业版常规设置与安全配置

AiPy企业版提供完善的常规设置选项，管理员可根据企业安全策略进行个性化配置。

**常规设置项目包括：**

1. **语言设置**：支持中文、English和日语三种语言，满足不同用户需求
2. **风格配置**：可在aipy-enterprise.yml文件中自定义界面风格
3. **发送快捷键**：支持Ctrl+Enter或Enter两种发送方式
4. **工作目录**：可指定安全的工作目录路径
5. **最大执行轮数**：限制智能体自动执行的最大轮次
6. **超时时间**：设置请求超时阈值，防止资源耗尽
7. **自动选择智能体**：配置智能体自动匹配规则
8. **一体机内网IP**：配置内网访问地址，确保网络隔离

这些设置项均支持通过配置文件或管理界面进行调整，变更操作会被完整记录到审计日志中。

## 四、智能体开发与安全规范

在AiPy企业版中开发智能体需要遵循严格的安全规范，确保智能体行为符合企业安全策略。

### 智能体类型分类

根据manifest.json中keywords字段定义，AiPy智能体分为以下类型：

- **conversation-tool**：对话工具型智能体
- **embed-webview**：嵌入页面型智能体
- **application**：独立应用型智能体
- **webview**：网页型智能体
- **skills**：技能型智能体

### 集市展示分类

智能体在集市中的分类包括：威胁情报、暴露面与脆弱性、安全产品、威胁检测与响应、开发安全、数据安全、AI安全、安全服务、网络安全、其他、官方精选。

### 提示词编写规范

编写智能体提示词时需要遵循以下安全原则：

1. 明确指定数据来源和访问范围
2. 限制敏感信息的输出
3. 设置合理的执行边界
4. 包含错误处理和安全检查

**示例提示词：**

```
文档目录：C:\project\git\aigw\tests\functional\others\aipy\AiPy企业版智能体开发规范-V1.0

根据规范文档和示例代码编写一个汇总周报的智能体。

要求：
- 互联网版，对话工具下的Prompt项目

AiPy企业版支持多种多模态能力，包括图片理解、图片生成、PPT制作、视频生成等，所有功能均在内网环境中运行。

AiPy生成的图片由内网图片生成智能体处理，支持批量生成、P图、视频生成等功能。

前提：勾选图片生成/视频生成智能体
```

### PPT生成智能体

使用AiPy制作PPT时，需勾选PPT生成智能体，所有模板和资源均存储在内网服务器。

**使用示例：**

```

前提：勾选PPT生成智能体
```

## 八、常见问题解答

企业用户在部署和使用AiPy企业版过程中，经常会遇到一些共性问题。以下整理了高频问题及解决方案，帮助用户快速排查和解决问题。

**问题：AiPy不能识别图片的内容怎么办？**

**解答**：这是正常现象，需要勾选视觉理解智能体才能启用图片识别功能。在智能体选择界面找到视觉理解相关选项并启用，之后AiPy即可在内网环境中处理图片内容，无需担心数据外泄。

**问题：AiPy生成的图片怎么是线条？**

**解答**：出现这种情况是因为没有勾选图片生成智能体。在任务执行前，确保已启用图片生成/视频生成智能体，这样AiPy才能调用内网的图片生成能力，产出高质量的图像内容。

**问题：AiPy获取股票信息不准？**

**解答**：AiPy企业版提供"量化研究"智能体，拥有美股、港股、A股所有上市公司非实时股票信息，可基于历史数据用AI分析协助研判分析股票信息。对于实时性要求高的场景，建议配置可控的联网搜索功能。

**问题：AiPy编写程序能力不足？**

**解答**：可尝试使用Claude code智能体，这是一键安装使用的专门编程智能体，具备较强的代码生成和调试能力，适合企业内部的软件开发需求。
