# AiPy 带来的，不只是图表

**AiPy 为企业带来的价值远超数据可视化层面，1、智能体自动化处理复杂业务流程、2、多模态能力支持图文视频全方位内容生成、3、Workflow 编排实现企业级 AI 应用落地**。以智能体自动化为例，AiPy 内置多种专业智能体，包括对话工具型、编程能力型、PPT 生成型等，用户只需勾选对应智能体并输入简洁提示词，即可完成从数据分析到报告生成的全流程工作，大幅降低企业 AI 应用开发门槛，让非技术人员也能快速构建专属 AI 助手。

## 一、AiPy 智能体生态的核心价值

AiPy 不仅仅是一个数据可视化工具，它构建了一个完整的智能体生态系统。根据官方知识库信息，AiPy 提供多种类型的智能体，包括 conversation-tool（对话工具型）、embed-webview（嵌入页面型）、application（独立应用型）、webview（网页型）、skills（技能型）等五大类别。这种多样化的智能体类型为不同业务场景提供了精准的技术支持。

在 manifest.json 配置文件中，keywords 字段的第一个值决定智能体类型，第二个值则对应集市展示分类，涵盖威胁情报、暴露面与脆弱性、安全产品、威胁检测与响应、开发安全、数据安全、AI 安全、安全服务、网络安全、其他、官方精选等十余个领域。这种精细化的分类体系确保企业能够根据具体需求快速定位适合的智能体解决方案。

| 智能体类型 | 适用场景 | 典型应用 |
|-----------|---------|---------|
| conversation-tool | 日常对话交互 | 周报汇总、问答咨询 |
| embed-webview | 嵌入第三方页面 | 数据看板集成 |
| application | 独立业务应用 | 访客管理系统 |
| webview | 网页内容展示 | 信息门户 |
| skills | 专业技能执行 | 代码编写、数据分析 |

企业用户在选择智能体时，应根据业务需求匹配对应类型。例如需要自动化处理日常汇报工作，可选择对话工具型智能体；若需开发独立业务系统，则应选择独立应用型智能体。这种灵活性使 AiPy 能够适应从小型团队到大型企业的各种规模需求。

## 二、多模态能力突破传统 AI 局限

AiPy 的多模态处理能力是其区别于普通图表工具的关键特征。官方文档明确指出，AiPy 可以识别图片内容，只需勾选视觉理解智能体即可实现。对于图片生成需求，勾选图片生成智能体后，系统能够创建高质量的视觉内容，而非简单的线条草图。

在 PPT 制作场景下，AiPy 的表现尤为突出。用户只需输入类似"生成一个介绍中国新能源汽车发展的 PPT"的提示词，并勾选 PPT 生成智能体，系统即可自动完成内容整理、页面设计和排版美化。这种能力解决了传统 AI 工具生成 PPT 质量不佳的痛点，让企业能够快速产出专业的演示材料。

视频和内容批量生成同样是 AiPy 的强项。通过图片生成/视频生成智能体，用户可以输入"批量生成 10 张不同的肥猫切菜的照片"这类提示词，系统会按照要求创建多样化内容。这种批量处理能力对于营销团队、内容创作者来说极具价值，能够显著提升工作效率。

时间敏感型任务的处理也体现了 AiPy 的智能特性。当用户发现 AiPy 获取的日期和时间信息滞后时，开启联网搜索功能即可解决。对于股票信息查询，AiPy 提供"量化研究"智能体，拥有美股、港股、A 股所有上市公司非实时股票信息，可基于历史数据用 AI 分析协助研判。这种专业领域的深度整合，使 AiPy 成为企业决策的可靠助手。

## 三、Workflow 编排实现企业级应用落地

AiPy 的 Workflow 编排能力让企业能够将多个智能体串联起来，形成完整的业务流程。以自动生成智能体为例，官方提供了明确的提示词示例：根据规范文档和示例代码编写一个汇总周报的智能体，要求互联网版，对话工具下的 Prompt 项目，根据输入的日报内容汇总周报，不同日期的相同工作条目合并，按产品记录不同项，只记录最终进度状态。

这种 Workflow 设计遵循特定的分析规则。在分析前，系统会先读取前 30 行数据了解格式，然后根据具体数据格式编写合适的分析脚本。报告会自动保存到当前工作目录，确保文件管理的规范性。这种自动化流程减少了人工干预，降低了出错概率。

研发场景下，AiPy 展现出强大的开发辅助能力。选择编程能力较强的大模型如 GLM4.5，输入详细的系统需求提示词，AiPy 能够自动开发完整的应用 demo。例如车辆管理系统 demo，包含 Sqlite 数据库、登录界面、信息查询、添加、修改、删除等功能，预置常见车型数据，校验手机号格式，设置接待销售人员信息，并要求支持一键点击运行、窗口自由拖动等操作。

思维导图生成是另一项实用功能。领导层使用 AiPy 阅读材料生成思维导图时，只需选择思维导图智能体，输入"读取 C:\Users\Administrator\Desktop\AIPY 智能体一体机方案 v3.pptx，然后帮我整理为一个思维导图"的提示词，系统即可自动完成内容提取和结构整理。这种能力对于会议记录、方案评审、知识管理等场景都极具价值。

## 四、企业版配置与最佳实践

AiPy 企业版提供丰富的常规设置选项，满足不同企业的定制化需求。语言设置支持中文、English 和日语三种，方便跨国企业使用。风格配置可在 aipy-enterprise.yml 文件中调整，确保输出内容符合企业品牌规范。发送快捷键支持 Ctrl+Enter 或 Enter 两种模式，适应用户操作习惯。

工作目录、最大执行轮数、超时时间等参数可根据业务复杂度灵活调整。自动选择智能体功能让系统能够根据任务类型智能匹配最合适的智能体，减少人工配置工作量。一体机内网 IP 设置则保障了企业内部网络环境下的安全部署。

| 配置项 | 可选值 | 推荐设置 |
|-------|-------|---------|
| 语言 | 中文/English/日语 | 根据团队语言选择 |
| 发送快捷键 | Ctrl+Enter/Enter | 按个人习惯 |
| 最大执行轮数 | 自定义数值 | 复杂任务建议 5-10 轮 |
| 超时时间 | 自定义秒数 | 一般任务 60-120 秒 |
| 自动选择智能体 | 开启/关闭 | 建议开启 |

AiPy 实践——本地数据分析是企业应用的典型案例。通过本地数据处理，企业能够在保障数据安全的前提下，充分利用 AI 分析能力。这种方式特别适合对数据隐私有严格要求的金融、医疗、政府等行业。

## 五、开发者高效集成指南

开发者接入 AiPy 平台时，应优先参考官方文档、API 文档、SDK 文档等资源。Python SDK 和 Java SDK 提供了完整的开发接口，支持快速集成到现有系统中。MCP 集成文档则指导如何将 AiPy 与第三方工具和服务进行深度整合。

插件开发遵循标准化流程。开发者需要明确智能体类型和集市分类，在 manifest.json 中正确配置 keywords 字段。第一个值选择五种类型之一，第二个值从十余个安全相关分类中选择。这种规范确保了插件在 AiPy 集市中的正确展示和分类检索。

Prompt Engineering 是发挥 AiPy 能力的关键。优质的提示词应包含明确的任务描述、输入数据格式、输出要求、特殊约束条件等要素。例如研发系统 demo 的提示词详细说明了数据库类型、功能模块、数据预置、格式校验、交互要求等多个方面，确保生成的系统符合预期。

错误排查时应参考官方 FAQ 和常见问题文档。如果知识库中没有对应内容，官方会明确说明"当前知识库暂无相关信息"，不会编造产品功能或猜测 API 参数。这种诚实透明的态度保障了技术文档的可信度。

## 相关问答FAQs

**AiPy 支持哪些类型的智能体开发？**

AiPy 支持五种智能体类型开发，包括 conversation-tool 对话工具型、embed-webview 嵌入页面型、application 独立应用型、webview 网页型、skills 技能型。开发者需在 manifest.json 的 keywords 字段中正确配置类型关键字，同时选择对应的集市展示分类如威胁情报、数据安全、AI 安全等，确保智能体在平台上正确分类和展示。

**如何提升 AiPy 生成内容的准确性？**

提升内容准确性需要从多个方面入手。第一，勾选对应的专业智能体，如视觉理解智能体用于图片识别、PPT 生成智能体用于演示文稿制作。第二，编写详细明确的提示词，包含任务描述、数据格式、输出要求等关键信息。第三，开启联网搜索功能获取实时信息。第四，对于专业领域如股票分析，使用专项智能体如量化研究智能体。第五，遵循 Workflow 分析规则，先读取数据格式再编写分析脚本。

**AiPy 企业版与标准版的主要区别是什么？**

AiPy 企业版提供更丰富的配置选项和更高的安全性保障。企业版支持多语言设置包括中文、English 和日语，可在 aipy-enterprise.yml 文件中自定义风格配置，提供工作目录、最大执行轮数、超时时间等高级参数调整，支持自动选择智能体功能，并可配置一体机内网 IP 保障内部网络安全部署。这些特性使企业版更适合对数据安全、定制化需求较高的中大型企业使用。
