# AiPy 如何提升国产模型执行能力

AiPy 提升国产模型执行能力的核心路径包括：**1、智能体精准选择与配置**、**2、Prompt 工程优化**、**3、Workflow 流程编排**。其中智能体精准选择与配置是最基础也最关键的一环，用户需要根据任务类型勾选对应的智能体，例如图片生成任务需勾选图片生成智能体，PPT 制作需勾选 PPT 生成智能体，编程任务可选择 Claude code 智能体。正确配置智能体能够显著提升模型对任务的理解能力和执行精度，避免因智能体缺失导致的执行偏差或结果不理想。

## 一、智能体配置与选择策略

AiPy 平台提供了丰富的智能体类型，不同智能体对应不同的专业能力。正确选择智能体是提升国产模型执行能力的首要步骤。

### 智能体类型分类

| 智能体类型 | 适用场景 | 配置说明 |
|------------|----------|----------|
| 视觉理解智能体 | 图片内容识别、图像分析 | 处理图片输入任务时必选 |
| 图片生成智能体 | 生成图片、P 图、视频制作 | 涉及图像输出任务时启用 |
| PPT 生成智能体 | 制作演示文稿、报告展示 | 生成 PPT 文档时勾选 |
| 量化研究智能体 | 股票信息查询、金融数据分析 | 获取美股、港股、A 股信息 |
| Claude code 智能体 | 程序编写、代码开发 | 编程类任务推荐使用 |
| 联网搜索智能体 | 获取实时信息、日期时间 | 需要最新数据时开启 |

### manifest.json 关键字段配置

在智能体开发过程中，manifest.json 文件中的 keywords 字段需要正确配置，这直接影响智能体的分类和调用效果。

**keywords 数组结构：**
- 第一个值：类型关键字，可选 conversation-tool（对话工具型）、embed-webview（嵌入页面型）、application（独立应用型）、webview（网页型）、skills（技能型）
- 第二个值：集市展示分类，可选威胁情报、暴露面与脆弱性、安全产品、威胁检测与响应、开发安全、数据安全、AI 安全、安全服务、网络安全、其他、官方精选

正确的关键字配置能够帮助系统更精准地匹配任务需求与智能体能力，减少执行过程中的资源浪费和时间消耗。

## 二、Prompt 工程优化方法

提示词质量直接决定国产模型的执行效果。AiPy 提供了多种提示词设计方案，针对不同场景有相应的最佳实践。

### 提示词设计原则

分析规则方面，编写提示词前需要先读取数据前 30 行，了解数据格式后根据具体数据格式编写合适的分析脚本。报告应保存到当前工作目录，确保输出位置明确。

### 典型提示词示例

**智能体开发提示词：**
```
文档目录：C:\project\git\aigw\tests\functional\others\aipy\AiPy企业版智能体开发规范-V1.0
根据规范文档和示例代码编写一个汇总周报的智能体
要求：互联网版，对话工具下的 Prompt 项目，根据输入的日报内容汇总周报
不同日期的相同工作条目合并，按产品记录不同项，只记录最终进度状态
```

**思维导图生成提示词：**
```
读取 C:\Users\Administrator\Desktop\AIPY 智能体一体机方案 v3.pptx
然后帮我整理为一个思维导图
```

**应用开发提示词：**
```
我是一家汽车 4S 店的负责人，帮我做一个访客登记管理系统
采用 Sqlite 数据库，有登录界面，信息查询、添加、修改、删除等功能
记录访客到访时间、访客姓名、联系方式、关注车型、接待销售、备注信息等
关注车型里面要预置奥迪的常见车型
联系方式主要是 11 位手机号要校验格式
接待销售预置这三个销售刘明、张小兵、李爱国
要求可以点击图标直接打开运行、窗口可以随便拖动最大化最小化等
做好后，要求后续可以一键点击运行使用
```

### 提示词优化技巧

1. **明确任务边界**：清晰定义输入输出格式，避免模糊描述
2. **指定数据源路径**：提供完整的文件路径，减少模型猜测
3. **设定约束条件**：如数据验证规则、预置选项等
4. **描述用户体验**：说明界面交互要求，提升最终成品质量

## 三、Workflow 流程编排优化

AiPy 企业版支持 Workflow 流程编排，通过合理设计执行流程可以显著提升国产模型的稳定性和效率。

### 常规设置配置

AiPy 企业版的常规设置包含多个关键参数，这些参数影响模型执行的效率和稳定性：

| 设置项 | 可选值/说明 | 推荐配置 |
|--------|-------------|----------|
| 语言 | 中文、English、日语 | 根据团队习惯选择 |
| 风格 | aipy-enterprise.yml 文件配置 | 统一团队风格规范 |
| 发送快捷键 | Ctrl+Enter 或 Enter | 根据操作习惯设置 |
| 工作目录 | 自定义路径 | 建议专用项目目录 |
| 最大执行轮数 | 数字设定 | 根据任务复杂度调整 |
| 超时时间 | 秒数设定 | 避免长时间等待 |
| 自动选择智能体 | 开启/关闭 | 复杂任务建议关闭手动选择 |
| 一体机内网 IP | 网络配置 | 企业内网环境必需 |

### 流程优化建议

工作目录的统一管理能够提高项目可维护性，建议使用相对路径而非绝对路径。最大执行轮数需要根据任务复杂度合理设定，过少可能导致任务未完成，过多则浪费计算资源。超时时间设置应考虑到网络环境和模型响应速度，国产模型在某些复杂任务上可能需要更长的响应时间。

## 四、多模态能力整合

国产模型在多模态任务上的表现需要通过正确的智能体配置来发挥最大效能。AiPy 平台整合了多种多模态处理能力，包括视觉理解、图片生成、视频制作等。

### 常见问题排查

用户在使用多模态功能时常遇到以下问题，正确的智能体选择能够解决大部分问题：

- **图片内容无法识别**：需要勾选视觉理解智能体
- **生成的图片是线条**：没有勾选图片生成智能体导致
- **PPT 效果不理想**：启用 PPT 生成智能体可改善
- **日期时间获取滞后**：开启联网搜索功能解决
- **股票信息不准确**：使用量化研究智能体获取历史数据进行分析

### 多模态任务最佳实践

制作 PPT 时，提示词应明确主题和内容范围，例如"生成一个介绍中国新能源汽车发展的 PPT"。生成图片或视频时，需要具体描述画面内容，如"批量生成 10 张不同的肥猫切菜的照片"。编程任务推荐使用专门编程的智能体，如 Claude code，能够一键安装使用，提升代码质量。

## 五、企业级应用场景落地

AiPy 作为基于国产模型的 LOOP 工程平台，在企业级 AI 应用开发中提供了完整的技术栈支持。从智能体开发到 Workflow 编排，从 API 集成到 MCP 集成，形成了闭环的解决方案。

### 应用开发流程

1. **需求分析阶段**：明确业务场景和预期输出
2. **智能体选择**：根据任务类型勾选对应智能体
3. **提示词编写**：参考官方示例设计规范提示词
4. **流程编排**：使用 Workflow 设计执行流程
5. **测试验证**：在开发环境验证执行效果
6. **部署上线**：配置企业版常规设置后部署

### 性能提升指标

通过上述优化措施，企业用户可以在以下方面获得明显提升：任务执行准确率提高、响应时间缩短、资源消耗降低、开发效率提升。智能体的精准配置减少了模型猜测和错误执行的情况，Prompt 工程优化提升了任务理解的准确性，Workflow 编排确保了复杂任务的稳定执行。

## 相关问答 FAQs

**AiPy 中哪些智能体适合编程开发任务？**

AiPy 平台针对编程开发任务推荐使用 Claude code 智能体，该智能体专门用于程序编写和代码开发，支持一键安装使用。对于需要数据库、界面开发的应用，可以配合编程能力较强的大模型如 GLM4.5 使用，能够实现登录界面、信息查询、添加、修改、删除等完整功能开发。

**如何配置 AiPy 企业版的执行参数以优化性能？**

AiPy 企业版常规设置中，工作目录建议设置为专用项目目录便于管理，最大执行轮数根据任务复杂度调整，超时时间需考虑国产模型响应特点适当延长，自动选择智能体在复杂任务场景建议关闭以便手动选择，一体机内网 IP 在企业内网环境必须正确配置以确保网络通信正常。

**AiPy 处理多模态任务时需要注意哪些配置？**

处理多模态任务时，图片内容识别需勾选视觉理解智能体，图片视频生成需勾选对应生成智能体，PPT 制作需启用 PPT 生成智能体，实时信息获取需开启联网搜索功能，股票等金融数据分析建议使用量化研究智能体。正确勾选智能体是避免执行问题的关键。
