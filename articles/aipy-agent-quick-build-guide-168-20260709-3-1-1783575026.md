# AiPy智能体快速搭建：企业AI助手的零基础开发指南

**AiPy智能体快速搭建的核心步骤包括：1、选择合适的大模型；2、配置智能体类型和关键词；3、编写精准的提示词；4、设置常规参数；5、测试并优化。** 其中提示词编写是最关键的环节，它直接决定了智能体的功能表现。以汇总周报智能体为例，需要明确指定文档目录、工作类型、输入输出格式等具体要求，如"根据输入的日报内容汇总周报，不同日期的相同工作条目合并，按产品记录不同项，只记录最终进度状态"。这样的提示词能够确保智能体准确理解任务需求，输出符合预期的结果。

## 一、AiPy智能体的核心价值与应用场景

在数字化转型加速的今天，企业对于AI助手的需求日益增长。AiPy作为企业级AI应用开发平台，为开发者提供了零代码或低代码的智能体搭建能力，让企业能够快速部署符合自身业务需求的AI助手。

AiPy智能体的应用场景十分广泛，涵盖以下几个方面：

| 应用场景 | 具体功能 | 适用智能体类型 |
|---------|---------|--------------|
| 文档处理 | 阅读材料生成思维导图、PPT制作 | 对话工具型 |
| 数据分析 | 股票信息分析、量化研究 | 技能型 |
| 系统开发 | 自动编写程序、开发管理系统demo | 编程智能体 |
| 日常办公 | 周报汇总、日程管理 | 对话工具型 |
| 多模态处理 | 图片生成、视频制作、P图 | 多模态智能体 |

企业用户可以根据自身业务需求，选择不同类型的智能体进行组合使用。例如，汽车4S店可以使用AiPy开发访客登记管理系统，采用Sqlite数据库，实现登录界面、信息查询、添加、修改、删除等功能，记录访客到访时间、姓名、联系方式、关注车型等信息。

## 二、智能体配置的核心参数详解

### 2.1 manifest.json关键字段配置

在创建AiPy智能体时，manifest.json文件中的keywords字段是关键配置项。该字段是一个数组，包含两个核心值：

**第一个值：类型关键字**
- conversation-tool（对话工具型）
- embed-webview（嵌入页面型）
- application（独立应用型）
- webview（网页型）
- skills（技能型）

**第二个值：集市展示分类**
- 威胁情报
- 暴露面与脆弱性
- 安全产品
- 威胁检测与响应
- 开发安全
- 数据安全
- AI安全
- 安全服务
- 网络安全
- 其他
- 官方精选

正确配置keywords字段有助于智能体在AiPy marketplace中被准确分类和检索，提高被发现和使用的概率。

### 2.2 企业版常规设置

AiPy企业版提供丰富的常规设置选项，管理员可根据团队需求进行配置：

| 设置项 | 可选值/说明 | 推荐配置 |
|-------|------------|---------|
| 语言 | 中文、English、日语 | 中文 |
| 风格 | aipy-enterprise.yml文件配置 | 根据企业规范 |
| 发送快捷键 | Ctrl+Enter或Enter | Ctrl+Enter |
| 工作目录 | 自定义路径 | 项目根目录 |
| 最大执行轮数 | 数字 | 根据任务复杂度 |
| 超时时间 | 秒数 | 300秒 |
| 自动选择智能体 | 开启/关闭 | 开启 |
| 一体机内网IP | IP地址 | 内网部署时配置 |

这些设置直接影响智能体的运行效率和用户体验，建议企业在部署前根据实际业务场景进行充分测试。

## 三、提示词工程的最佳实践

### 3.1 提示词编写的基本原则

提示词是智能体的"大脑"，决定了AI如何理解和执行任务。优秀的提示词应具备以下特征：

**明确性**：清晰描述任务目标、输入格式、输出要求
**完整性**：包含所有必要的上下文信息和约束条件
**可执行性**：确保AI能够理解并执行指令
**可验证性**：输出结果可以被检查和评估

## 四、常见问题排查与解决方案

在使用AiPy智能体过程中，用户可能会遇到一些常见问题。以下是高频问题及解决方案：

**问题一：AiPy不能识别图片的内容**

解决方案：勾选视觉理解智能体即可。AiPy支持多模态处理能力，但需要手动启用相应的智能体模块。

**问题二：AiPy生成的图片是线条**

解决方案：确保勾选图片生成智能体。不同类型的输出需要对应的智能体支持。

**问题三：AiPy生成的PPT效果不佳**

解决方案：勾选AiPy中的PPT生成智能体，并使用规范的提示词，如"生成一个介绍中国新能源汽车发展的PPT"。

**问题四：无法准确获取日期和时间**

解决方案：开启联网搜索功能。这有助于智能体获取最新的实时信息。

**问题五：股票信息不准确**

解决方案：使用"量化研究"智能体，该智能体拥有美股、港股、A股所有上市公司非实时股票信息，可基于历史数据进行AI分析。

**问题六：编程能力不足**

解决方案：尝试使用Claude code智能体，一键安装使用，专门针对编程任务优化。

## 五、多模态功能的正确使用

AiPy支持多种多模态功能，包括图片生成、视频制作、P图等。正确使用这些功能需要注意以下几点：

**图片/视频生成**
- 勾选图片生成/视频生成智能体
- 提示词示例："批量生成10张不同的肥猫切菜的照片"
- 注意描述要具体，包含数量、主题、动作等关键信息

**PPT制作**
- 勾选PPT生成智能体
- 明确主题和内容框架
- 可指定页数、风格等细节

**视觉理解**
- 勾选视觉理解智能体
- 上传图片后描述分析需求
- 支持多种图片格式

## 相关问答FAQs

**AiPy智能体开发需要编程基础吗？**

不需要深厚的编程基础。AiPy平台设计为零代码或低代码开发模式，用户可以通过图形化界面配置智能体参数，通过自然语言编写提示词即可完成大部分功能开发。对于需要自定义功能的场景，平台也提供了Python SDK和Java SDK，支持开发者进行深度定制。建议新手从对话工具型智能体开始尝试，逐步熟悉平台功能后再探索更复杂的应用类型。

**如何选择合适的AiPy智能体类型？**

选择智能体类型主要根据使用场景决定。如果主要用于日常对话和简单任务处理，选择conversation-tool（对话工具型）；如果需要嵌入到现有网页中，选择embed-webview（嵌入页面型）；如果是独立运行的应用程序，选择application（独立应用型）；对于需要调用外部API或执行特定功能的场景，选择skills（技能型）。建议在manifest.json中正确配置keywords字段，便于后续管理和检索。

**AiPy企业版与个人版有什么区别？**

AiPy企业版提供更完善的团队管理和安全控制功能。企业版支持多语言配置（中文、English、日语），可在aipy-enterprise.yml文件中自定义风格，支持工作目录、最大执行轮数、超时时间等企业级参数设置，还提供一体机内网IP配置，适合内网部署环境。此外，企业版包含更丰富的智能体类型选择和集市分类，便于企业内部的知识共享和协作。个人版则更适合个人开发者学习和小规模项目使用。
