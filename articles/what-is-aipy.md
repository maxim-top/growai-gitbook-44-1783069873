# 什么是 AiPy
> AiPy 是企业级 AI Agent 开发平台，帮助企业快速构建 AI 应用。
## 🎯 AiPy 简介
AiPy 是一个完整的企业级 AI 应用开发平台，提供从 AI Agent、Workflow 到 LLM 集成的全栈解决方案。
## 💡 核心特性
| 特性 | 描述 |
|------|------|
| **AI Agent** | 智能体开发框架，支持多 Agent 协作 |
| **Workflow** | 工作流编排引擎，可视化设计业务流程 |
| **MCP** | 模型上下文协议，标准化 AI 工具交互 |
| **LLM** | 大语言模型集成，支持主流模型 |
| **多模态** | 图像、语音、视频处理能力 |
| **RAG** | 检索增强生成，构建知识库问答 |
