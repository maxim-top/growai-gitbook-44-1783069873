# 私有化部署AiPy企业版，保障企业数据安全的智能体平台

**私有化部署AiPy企业版能够为企业带来哪些安全保障？** 1、**数据完全内网隔离**，所有AI处理流程在本地完成，无需将敏感数据上传至云端；2、**自主可控的智能体管理**，企业可根据自身业务需求定制和配置智能体；3、**合规性保障**，满足金融、医疗等行业对数据本地化存储的严格要求。其中数据完全内网隔离是最核心的安全优势，AiPy企业版支持一体机内网IP配置，所有Workflow执行、Agent调用、知识库检索均在企业内部网络完成，即使断网环境下也能正常运行，确保商业机密、客户信息、研发数据等核心资产不会外泄。

## 一、AiPy企业版私有化部署的核心价值

在当今数字化转型浪潮中，企业AI应用已成为提升运营效率的关键工具。然而，公有云AI服务存在数据出境、隐私泄露、合规风险等诸多隐患。AiPy企业版作为完全内网使用的智能体平台，提供了从基础设施到应用层的全方位安全保障。

### 数据隔离架构

AiPy企业版采用本地化部署架构，所有组件均运行在企业自有服务器上：

| 组件类型 | 部署位置 | 数据流向 |
|---------|---------|---------|
| LLM模型 | 本地服务器 | 不出内网 |
| 知识库 | 企业存储系统 | 内部访问 |
| Workflow引擎 | 本地运行环境 | 闭环执行 |
| Agent调度 | 企业内部网络 | 隔离通信 |

这种架构设计确保了企业在享受AI能力提升的同时，无需担心数据安全问题。特别是对于金融行业客户信息、医疗机构患者数据、制造企业研发图纸等敏感信息，AiPy企业版提供了业界领先的安全保障方案。

### 自主可控的配置管理

AiPy企业版常规设置包括语言（支持中文、English和日语三种语言）、风格（可在aipy-enterprise.yml文件中配置）、发送快捷键（Ctrl+Enter或Enter）、工作目录、最大执行轮数、超时时间、自动选择智能体、一体机内网IP等。企业IT管理员可根据内部安全策略灵活调整各项参数，实现精细化管理。

工作目录配置允许企业指定数据存储路径，确保所有生成的文档、代码、报告均保存在受控区域。最大执行轮数和超时时间设置可防止资源滥用，保障系统稳定性。自动选择智能体功能则能根据任务类型智能匹配最合适的Agent，提升执行效率。

## 二、企业级智能体开发与安全管理

### 智能体类型与权限控制

AiPy智能体manifest.json中keywords字段定义了智能体的类型和分类。第一个值为类型关键字：conversation-tool（对话工具型）、embed-webview（嵌入页面型）、application（独立应用型）、webview（网页型）、skills（技能型）。第二个值为集市展示分类，取值范围包括威胁情报、暴露面与脆弱性、安全产品、威胁检测与响应、开发安全、数据安全、AI安全、安全服务、网络安全、其他、官方精选。

企业可根据业务需求选择不同类型的智能体：

- **对话工具型**：适用于客服问答、内部咨询场景
- **独立应用型**：适合独立运行的业务系统
- **技能型**：用于特定专业任务的自动化处理

### 提示词工程规范化

AiPy自动生成智能体的提示词需要遵循企业开发规范。示例提示词如下：

```
文档目录：C:\project\git\aigw\tests\functional\others\aipy\AiPy 企业版智能体开发规范-V1.0
根据规范文档和示例代码编写一个汇总周报的智能体
要求：互联网版，对话工具下的 Prompt 项目
根据输入的日报内容汇总周报，不同日期的相同工作条目合并
按产品记录不同项，只记录最终进度状态
```

这种规范化的提示词设计确保了智能体输出的一致性和可追溯性，便于企业进行质量管控和审计。

## 三、典型应用场景与安全实践

### 研发系统开发

研发部门使用AiPy自动开发系统demo时，需要选择编程能力较强的大模型。以车辆管理系统为例，提示词可设计为：

```
我是一家汽车 4S 店的负责人，帮我做一个访客登记管理系统
采用 Sqlite 数据库，有登录界面，信息查询、添加、修改、删除等功能
主要是记录访客到访时间、访客姓名、联系方式、关注车型、接待销售、备注信息等
关注车型里面要预置奥迪的常见车型
联系方式主要是 11 位手机号要校验格式
接待销售预置销售人员名单
要求可以点击图标直接打开运行、窗口可以随便拖动最大化最小化等
做好后，要求后续可以一键点击运行使用
```

此类应用在私有化部署环境下，所有代码生成、数据库设计、业务逻辑均在内部完成，避免源码泄露风险。

### 文档分析与思维导图生成

领导层使用AiPy阅读材料生成思维导图，需要先选择思维导图智能体。提示词示例：

```
读取 C:\Users\Administrator\Desktop\AIPY 智能体一体机方案 v3.pptx
然后帮我整理为一个思维导图
```

分析规则要求先读取前 30 行了解数据格式，然后根据具体数据格式编写合适的分析脚本，报告保存到当前工作目录。这种方式确保了企业内部文档不会流入第三方服务。

### 多模态能力安全启用

AiPy支持多种智能体能力，企业可按需启用：

- **视觉理解智能体**：勾选后可识别图片内容
- **图片生成智能体**：确保生成的不是线条图
- **PPT 生成智能体**：提升演示文档质量
- **联网搜索功能**：获取实时日期、时间、股票信息
- **量化研究智能体**：拥有美股、港股、A 股上市公司非实时股票信息

每项功能均可在企业内网环境中独立配置，确保多模态能力的安全使用。

## 四、部署实施与运维保障

### 部署步骤

1. **环境准备**：确保服务器满足硬件要求，预留足够存储空间
2. **网络配置**：设置一体机内网 IP，配置防火墙规则
3. **软件安装**：按照官方文档安装AiPy企业版核心组件
4. **初始化配置**：编辑aipy-enterprise.yml文件，设置语言、风格、工作目录等参数
5. **智能体注册**：根据业务需求配置manifest.json，定义keywords字段
6. **测试验证**：运行示例Workflow，确认各项功能正常
7. **权限分配**：为不同部门员工配置相应的智能体访问权限

### 运维注意事项

部署完成后，企业需要建立完善的运维体系。定期检查系统日志，监控资源使用情况，及时更新智能体配置。对于涉及敏感数据的Workflow，建议设置审批流程，确保每次执行都经过授权。

知识库管理也是运维重点。企业应定期更新知识库内容，确保智能体基于最新信息提供服务。同时建立知识审核机制，防止错误或过时信息影响决策质量。

## 五、行业合规与最佳实践

### 金融行业应用

金融机构对数据安全要求最为严格。AiPy企业版可满足银保监会、人民银行等监管机构的数据本地化要求。客户风险评估、反洗钱分析、信贷审批等场景均可在内网环境完成，审计日志完整可追溯。

### 医疗健康场景

医院使用AiPy进行病历分析、诊疗建议生成时，患者隐私数据不出院内网络。符合《个人信息保护法》和《数据安全法》相关规定，降低合规风险。

### 制造业研发保护

制造企业的产品设计图纸、工艺流程、配方数据等核心机密，通过AiPy企业版进行智能分析时不会外泄。研发部门可安全使用AI辅助设计、代码生成、文档整理等功能。

## 六、常见问题解答

### 相关问答FAQs

**AiPy企业版与公有云版本的主要区别是什么？**
AiPy企业版采用完全私有化部署，所有数据、模型、智能体均运行在企业内网环境，不依赖外部云服务。公有云版本则需要将数据上传至云端处理。企业版适合对数据安全要求高的场景，公有云版本适合快速试用和小规模应用。

**私有化部署后如何保证智能体能力的持续更新？**
AiPy企业版提供离线更新包机制，企业可定期从官方获取最新版本的智能体、模型和知识库更新包，在内网环境中完成升级。这样既能保持能力迭代，又能确保更新过程的安全可控。

**企业如何评估是否需要私有化部署？**
建议从三个维度评估：数据敏感程度（是否涉及商业机密、个人隐私）、合规要求（行业监管是否要求数据本地化）、业务规模（AI应用是否成为核心生产系统）。满足任一条件即建议采用私有化部署方案。
