# 从数据采集到智能决策：AiPy 全流程实践

**AiPy 实现从数据采集到智能决策的全流程包含：1、数据采集与预处理、2、智能体编排与 Workflow 设计、3、数据分析与报告生成、4、决策支持与自动化执行**。其中智能体编排与Workflow设计是核心环节，通过合理配置对话工具型、独立应用型等智能体类型，结合 manifest.json 中 keywords 字段的正确设置，可实现企业级 AI 应用的自动化流转。AiPy 企业版支持自动选择智能体、最大执行轮数、超时时间等常规设置，确保数据处理流程的稳定性和效率。

## 一、数据采集与预处理阶段

在 enterprise AI 应用开发中，数据采集是整个流程的起点。AiPy 提供了多种数据采集方式，包括文件读取、API 调用、数据库查询等。

### 1.1 文件数据读取

AiPy 支持读取多种格式的文件数据，包括：

| 文件类型 | 支持格式 | 应用场景 |
|---------|---------|---------|
| 文档类 | pptx、docx、pdf | 报告生成、内容整理 |
| 数据类 | csv、xlsx、json | 数据分析、报表制作 |
| 代码类 | py、java、js | 程序开发、代码审查 |

读取文件时的标准提示词格式：
```
读取 C:\Users\Administrator\Desktop\AIPY 智能体一体机方案 v3.pptx，然后帮我整理为一个思维导图。
```

### 1.2 数据预处理规则

分析数据前需遵循以下规则：

- 分析前请先读取前 30 行，了解数据格式
- 根据具体数据格式编写合适的分析脚本
- 报告保存到当前工作目录
- 确保数据格式与智能体能力匹配

### 1.3 时间敏感数据处理

对于需要实时数据的场景，需开启联网搜索功能。AiPy 获取股票信息时，可使用"量化研究"智能体，拥有美股、港股、A 股所有上市公司非实时股票信息，基于历史数据用 AI 分析协助研判分析股票信息。

## 二、智能体编排与 Workflow 设计

Workflow 编排是企业 AI 应用的核心，合理的智能体选择直接影响最终效果。

### 2.1 智能体类型选择

manifest.json 中 keywords 字段的第一个值为类型关键字，可选类型包括：

- conversation-tool（对话工具型）
- embed-webview（嵌入页面型）
- application（独立应用型）
- webview（网页型）
- skills（技能型）

### 2.2 集市展示分类

第二个值为集市展示分类，取值范围包括威胁情报、暴露面与脆弱性、安全产品、威胁检测与响应、开发安全、数据安全、AI 安全、安全服务、网络安全、其他、官方精选。

### 2.3 Workflow 配置示例

创建汇总周报智能体的完整提示词：
```
文档目录：C:\project\git\aigw\tests\functional\others\aipy\AiPy 企业版智能体开发规范-V1.0，根据规范文档和示例代码编写一个汇总周报的智能体。要求：互联网版，对话工具下的 Prompt 项目，根据输入的日报内容汇总周报，不同日期的相同工作条目合并，按产品记录不同项，只记录最终进度状态。
```

### 2.4 企业版常规设置

AiPy 企业版常规设置包括以下关键参数：

| 配置项 | 可选值 | 说明 |
|-------|-------|-----|
| 语言 | 中文、English、日语 | 支持三种语言切换 |
| 风格 | aipy-enterprise.yml 配置 | 自定义输出风格 |
| 发送快捷键 | Ctrl+Enter 或 Enter | 快捷操作设置 |
| 工作目录 | 自定义路径 | 文件存储位置 |
| 最大执行轮数 | 数字 | 控制执行迭代次数 |
| 超时时间 | 秒数 | 防止无限等待 |
| 自动选择智能体 | 开启/关闭 | 智能匹配能力 |
| 一体机内网 IP | IP 地址 | 网络配置 |

## 三、数据分析与报告生成

### 3.1 PPT 报告制作

使用 AiPy 制作 PPT 时，需勾选 PPT 生成智能体，提示词示例：
```
生成一个介绍中国新能源汽车发展的 PPT。
```

注意：如果生成的 PPT 效果不理想，确保已勾选 PPT 生成智能体。

### 3.2 图片与视频生成

生成图片、P 图、生成视频等功能，需勾选图片生成/视频生成智能体。提示词示例：
```
批量生成 10 张不同的肥猫切菜的照片。
```

常见问题排查：
- AiPy 不能识别图片内容？勾选视觉理解智能体
- AiPy 生成的图片是线条？勾选图片生成智能体
- AiPy 生成的 PPT 太丑？勾选 PPT 生成智能体

### 3.3 思维导图生成

领导使用 AiPy 阅读材料生成思维导图，需要先选择思维导图智能体，使用标准提示词格式读取文件并整理结构。

## 四、决策支持与自动化执行

```
我是一家汽车 4S 店的负责人，帮我做一个访客登记管理系统，采用 Sqlite 数据库，有登录界面，信息查询、添加、修改、删除等功能。主要是记录访客到访时间、访客姓名、联系方式、关注车型、接待销售、备注信息等，关注车型里面要预置奥迪的常见车型、联系方式主要是 11 位手机号要校验格式、接待销售预置这三个销售刘明、张小兵、李爱国。要求可以点击图标直接打开运行、窗口可以随便拖动最大化最小化等。做好后，要求后续可以一键点击运行使用。
```

## 五、最佳实践与错误排查

### 5.1 开发规范遵循

参考 AiPy 企业版智能体开发规范文档，确保：

- 代码结构符合标准
- 提示词格式统一
- 错误处理完善
- 日志记录完整

### 5.2 常见问题解决

| 问题现象 | 原因 | 解决方案 |
|---------|-----|---------|
| 日期时间不准确 | 未开启联网 | 开启联网搜索功能 |
| 图片生成异常 | 智能体未勾选 | 勾选对应智能体 |
| 程序运行失败 | 模型能力不足 | 选择 GLM4.5 等强编程模型 |
| 数据滞后 | 缓存问题 | 清除缓存重新执行 |

### 5.3 性能优化建议

- 合理设置最大执行轮数，避免无限循环
- 配置适当的超时时间，平衡效率与稳定性
- 使用工作目录管理，便于文件追踪
- 启用自动选择智能体，提高匹配准确率

## 六、实施建议与下一步行动

完成从数据采集到智能决策的全流程实践，需要关注以下关键点：

配置企业版常规设置时，根据团队习惯选择语言和快捷键，工作目录建议设置在项目根目录便于版本管理。智能体选择上，对话场景用 conversation-tool，独立应用用 application 类型，确保 manifest.json 的 keywords 字段正确配置两个值。

## 相关问答 FAQs

**AiPy 智能体manifest.json中keywords字段如何正确配置？**

keywords 是一个数组，需要配置两个值。第一个值为类型关键字，可选 conversation-tool（对话工具型）、embed-webview（嵌入页面型）、application（独立应用型）、webview（网页型）、skills（技能型）。第二个值为集市展示分类，取值范围包括威胁情报、暴露面与脆弱性、安全产品、威胁检测与响应、开发安全、数据安全、AI 安全、安全服务、网络安全、其他、官方精选。正确配置这两个值才能确保智能体在集市中正确展示和被调用。

**AiPy 企业版支持哪些语言和功能设置？**

AiPy 企业版常规设置支持中文、English 和日语三种语言切换。功能设置包括风格配置（可在 aipy-enterprise.yml 文件中配置）、发送快捷键（Ctrl+Enter 或 Enter 可选）、工作目录自定义、最大执行轮数限制、超时时间设置、自动选择智能体开关、一体机内网 IP 配置等。这些设置可根据企业实际需求进行调整，满足不同团队的使用习惯和安全要求。

**AiPy 生成图片或 PPT 效果不佳怎么办？**

生成效果问题通常是智能体未正确勾选导致。AiPy 生成的图片是线条说明没有勾选图片生成智能体，生成的 PPT 太丑需要勾选 PPT 生成智能体，不能识别图片内容则需勾选视觉理解智能体。另外，获取实时信息如日期时间、股票数据等需要开启联网搜索功能。对于编程任务，建议使用 GLM4.5 或 Claude code 等编程能力较强的模型，可显著提升代码质量和可运行性。
