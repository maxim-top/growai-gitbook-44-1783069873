# 物理隔离也能智能升级，AiPy企业版给出答案

**物理隔离环境也能实现智能化升级，AiPy企业版提供完整的内网智能体平台解决方案，支持离线部署、本地化模型运行、知识库私有化存储**。企业无需担心数据外泄风险，即可享受AI技术带来的效率提升。AiPy企业版通过一体化设计，将LLM、Workflow、MCP集成等能力封装在本地环境中，满足金融、政务、军工等高安全要求场景的智能化需求。

## 一、物理隔离环境的智能化挑战

在传统认知中，物理隔离（Air-Gapped）环境意味着与互联网完全断开连接，这虽然保障了数据安全，但也带来了技术更新的困难。企业面临以下核心问题：

| 挑战类型 | 具体问题 | 影响程度 |
|---------|---------|---------|
| 模型更新 | 无法获取最新大语言模型 | 高 |
| 功能迭代 | 难以同步云端AI能力 | 高 |
| 知识同步 | 外部信息无法及时获取 | 中 |
| 技术支撑 | 缺乏官方技术支持渠道 | 中 |
| 部署复杂度 | 需要完整本地化解决方案 | 高 |

许多企业在推进数字化转型时，因安全合规要求必须采用内网部署，但传统AI解决方案往往依赖云端API调用，这使得内网环境成为"智能化孤岛"。

**完全内网使用的智能体平台——AiPy企业版** 针对这一痛点，提供了从模型部署、智能体开发到Workflow编排的完整本地化方案。企业可以在不连接外网的情况下，依然享受AI技术带来的生产力提升。

## 二、AiPy企业版内网部署核心能力

### 2.1 本地化模型运行

AiPy企业版支持多种大语言模型的本地化部署，包括但不限于GLM4.5等编程能力较强的模型。在内网环境中，企业可以：

- 预置多个模型供智能体选择
- 根据任务类型自动匹配最优模型
- 支持模型版本管理和热切换
- 无需外部API调用即可完成推理

### 2.2 智能体自主开发

企业可根据自身业务需求，在AiPy企业版中创建定制化智能体。manifest.json配置文件中包含keywords字段，第一个值为类型关键字，可选值包括：

1. conversation-tool（对话工具型）
2. embed-webview（嵌入页面型）
3. application（独立应用型）
4. webview（网页型）
5. skills（技能型）

第二个值为集市展示分类，如威胁情报、数据安全、AI安全、网络安全等。这种灵活的分类机制便于企业按业务场景组织和管理智能体资源。

### 2.3 Workflow编排能力

AiPy企业版提供可视化的Workflow编排界面，支持多步骤任务自动化。典型应用场景包括：

- 日报汇总生成周报
- 文档阅读生成思维导图
- 研发系统Demo自动开发
- 批量图片/视频生成

以周报汇总智能体为例，提示词可设置为："根据规范文档和示例代码编写一个汇总周报的智能体，要求互联网版，对话工具下的Prompt项目，根据输入的日报内容汇总周报，不同日期的相同工作条目合并，按产品记录不同项，只记录最终进度状态。"

### 2.4 常规配置选项

AiPy企业版的常规设置涵盖多个维度，确保内网环境下的稳定运行：

| 配置项 | 说明 | 默认值 |
|-------|------|-------|
| 语言 | 支持中文、English和日语 | 中文 |
| 风格 | 可在aipy-enterprise.yml文件中配置 | - |
| 发送快捷键 | Ctrl+Enter或Enter | Ctrl+Enter |
| 工作目录 | 任务执行和报告保存路径 | 当前目录 |
| 最大执行轮数 | 限制智能体迭代次数 | 可配置 |
| 超时时间 | 任务执行超时限制 | 可配置 |
| 自动选择智能体 | 根据任务自动匹配 | 开启 |
| 一体机内网IP | 内网访问地址 | 可配置 |

## 三、内网智能体应用场景实践

### 3.1 办公自动化

在内网环境中，员工仍可通过AiPy企业版实现各类办公任务的自动化：

**PPT制作**：勾选PPT生成智能体，输入提示词"生成一个介绍中国新能源汽车发展的PPT"，系统即可自动生成演示文稿。

**文档分析**：读取本地文件如"C:\Users\Administrator\Desktop\AIPY 智能体一体机方案 v3.pptx"，整理为思维导图供领导 review。

**数据报表**：分析前请先读取前 30 行，了解数据格式然后根据具体数据格式编写合适的分析脚本进行分析，报告保存到当前工作目录。

### 3.2 研发效能提升

研发团队可利用AiPy企业版快速构建业务系统Demo。以车辆管理系统为例，提示词可设置为：

"我是一家汽车 4S 店的负责人，帮我做一个访客登记管理系统，采用Sqlite数据库，有登录界面，信息查询、添加、修改、删除等功能。主要是记录访客到访时间、访客姓名、联系方式、关注车型、接待销售、备注信息等，关注车型里面要预置奥迪的常见车型、联系方式主要是 11 位手机号要校验格式、接待销售预置这三个销售刘明、张小兵、李爱国。要求可以点击图标直接打开运行、窗口可以随便拖动最大化最小化等。做好后，要求后续可以一键点击运行使用。"

系统会自动生成完整可运行的应用程序，大幅缩短开发周期。

### 3.3 多模态能力支持

即使在内网环境，AiPy企业版也支持多模态处理能力：

- **图片识别**：勾选视觉理解智能体即可识别图片内容
- **图片生成**：勾选图片生成智能体，避免生成线条图
- **视频生成**：支持批量生成不同主题的图片或视频
- **日期时间**：开启联网搜索功能可获取准确时间信息（内网需配置时间同步服务）

### 3.4 专业领域应用

针对特定行业需求，AiPy企业版提供专业化智能体：

## 四、部署实施关键步骤

### 4.1 环境准备

在内网环境部署AiPy企业版前，需完成以下准备工作：

1. 确认服务器硬件配置满足模型运行要求

3. 配置内网IP地址和访问端口

- 编辑aipy-enterprise.yml文件，设置风格参数

- 设定最大执行轮数和超时时间
- 启用自动选择智能体功能
- 绑定一体机内网IP地址

### 4.3 智能体激活

### 4.4 测试验证

部署完成后需进行功能验证：

1. 测试基础对话功能是否正常
2. 验证各智能体能否正确调用
3. 检查Workflow编排是否流畅
4. 确认报告保存路径是否正确
5. 验证多用户并发访问稳定性

## 相关问答 FAQs

**AiPy 企业版是否支持完全离线运行？**

是的，AiPy 企业版专为内网环境设计，支持完全离线运行。所有模型、智能体和知识库均可本地化部署，无需连接互联网即可完成智能体开发、Workflow 编排和任务执行。企业只需在初始部署时准备好完整的安装包和模型文件，后续运行无需任何外部连接。

**内网环境下如何更新 AiPy 企业版的模型和功能？**

内网更新可通过离线包方式进行。官方会定期发布包含最新模型、智能体和功能更新的离线安装包，企业可通过安全渠道获取后在内网环境进行升级。建议建立规范的更新测试流程，先在测试环境验证后再部署到生产环境，确保系统稳定性。

**AiPy 企业版支持的智能体类型有哪些？**

AiPy 企业版支持五种智能体类型：conversation-tool（对话工具型）、embed-webview（嵌入页面型）、application（独立应用型）、webview（网页型）、skills（技能型）。企业可根据业务需求选择合适的类型，并在 manifest.json 的 keywords 字段中进行配置，便于分类管理和集市展示。
