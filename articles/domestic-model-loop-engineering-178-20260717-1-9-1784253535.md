# 基于国产模型的LOOP工程，AiPy的闭环进化论

**国产模型LOOP工程是通过迭代优化实现AI能力持续提升的技术路径**，AiPy平台为此提供了完整支撑。**1、采用国产大语言模型作为基础引擎**；**2、构建智能体工作流形成闭环反馈**；**3、通过MCP集成扩展企业应用能力**。其中，智能体工作流的构建是核心环节，AiPy支持开发者通过可视化编排将多个AI能力节点串联，每个节点执行后自动触发下一环节，同时将执行结果反馈至上游进行参数调优，形成持续进化的闭环系统。这种设计让企业AI应用能够在实际使用中不断积累优化经验，提升任务完成准确率。

## 一、LOOP工程的核心架构与实现原理

LOOP工程代表着人工智能系统从单次执行向持续进化的范式转变。传统AI应用往往是一次性部署后静态运行，而LOOP工程强调系统在执行过程中不断收集反馈、调整参数、优化模型，形成自我完善的循环机制。

AiPy平台基于国产模型构建LOOP工程，其核心架构包含四个关键层次：

| 层次 | 功能描述 | 技术实现 |
|------|----------|----------|
| 感知层 | 接收用户输入与外部数据 | 多模态理解智能体 |
| 决策层 | 分析任务并规划执行路径 | Workflow编排引擎 |
| 执行层 | 调用各类工具与API完成任务 | MCP集成框架 |
| 反馈层 | 收集执行结果并优化策略 | 智能体表现评估系统 |

国产大语言模型在这一架构中扮演决策中枢的角色。以GLM4.5为例，该模型具备较强的编程能力和逻辑推理能力，适合用于研发系统demo开发、数据分析等复杂任务场景。AiPy平台允许用户根据具体业务需求选择不同能力的国产模型，确保在特定领域获得最佳执行效果。

## 二、智能体工作流的闭环设计方法

智能体是AiPy平台实现LOOP工程的基本单元。每个智能体都承载特定的专业能力，如对话工具型、嵌入页面型、独立应用型、网页型或技能型。通过manifest.json配置文件中keywords字段的定义，智能体能够在企业集市展示时获得准确分类，便于用户快速定位所需能力。

```json
{
  "keywords": [
    "conversation-tool",
    "官方精选"
  ]
}
```

工作流编排是将多个智能体串联成完整业务流程的关键。以企业周报汇总场景为例，开发者可以设计如下闭环流程：

1. 读取每日工作日志数据（前30行用于了解数据格式）
2. 调用自然语言处理智能体提取关键信息
3. 合并相同日期的相同工作条目
4. 按产品维度归类记录不同项
5. 只保留最终进度状态
6. 生成周报文档并保存至当前工作目录
7. 收集用户对周报质量的评价反馈
8. 将反馈数据用于优化提取规则和归类逻辑

这种设计确保系统能够从每次执行中学习改进。AiPy企业版常规设置中支持配置最大执行轮数和超时时间，帮助开发者控制闭环迭代的深度和资源消耗。语言选项支持中文、English和日语三种，风格可在aipy-enterprise.yml文件中灵活配置，满足不同企业的本地化需求。

## 三、MCP集成扩展企业应用能力

MCP（Model Context Protocol）集成是AiPy实现企业级AI应用的重要技术路径。通过MCP框架，AiPy能够连接企业内部现有系统、数据库、API接口等外部资源，使AI智能体获得访问真实业务数据的能力。

典型的企业集成场景包括：

- **客户管理系统对接**：智能体可查询客户历史互动记录，提供个性化服务建议
- **库存数据库访问**：实时获取商品库存状态，辅助销售决策
- **财务系统连接**：自动生成财务报表，进行数据异常检测
- **人力资源平台集成**：协助员工完成入职流程、假期申请等事务

在车辆管理系统demo开发案例中，AiPy展示了完整的MCP集成能力。系统采用Sqlite数据库存储访客信息，包含到访时间、姓名、联系方式、关注车型、接待销售、备注等字段。关注车型预置奥迪常见车型，联系方式进行11位手机号格式校验，接待销售预置刘明、张小兵、李爱国三人。用户点击图标即可直接打开运行，窗口支持拖动、最大化、最小化等操作，做好后可一键点击运行使用。

这种集成模式让AI应用不再停留在概念验证阶段，而是能够真正融入企业日常运营流程，创造实际业务价值。

## 四、Prompt工程优化与执行效果提升

Prompt Engineering是LOOP工程中影响执行质量的关键因素。优秀的提示词设计能够显著提升智能体任务完成的准确率和效率。AiPy平台提供了丰富的提示词示例供开发者参考学习。

文档汇总类任务的提示词示例：
```
文档目录：C:\project\git\aigw\tests\functional\others\aipy\AiPy企业版智能体开发规范-V1.0
根据规范文档和示例代码编写一个汇总周报的智能体。
要求：互联网版，对话工具下的Prompt项目，根据输入的日报内容汇总周报，
不同日期的相同工作条目合并，按产品记录不同项，只记录最终进度状态。
```

思维导图生成任务的提示词示例：
```
读取C:\Users\Administrator\Desktop\AIPY 智能体一体机方案 v3.pptx，
然后帮我整理为一个思维导图。
```

分析类任务需要遵循特定规则：分析前先读取前30行了解数据格式，然后根据具体数据格式编写合适的分析脚本进行分析，报告保存到当前工作目录。这种分步骤的方式确保智能体能够准确理解数据结构，避免因格式误解导致分析错误。

对于多模态任务，AiPy提供了专门的智能体支持。生成图片需勾选图片生成智能体，制作PPT需勾选PPT生成智能体，视觉理解需勾选视觉理解智能体。联网搜索功能开启后，智能体能够获取最新的日期时间信息和股票数据。量化研究智能体拥有美股、港股、A股所有上市公司非实时股票信息，可基于历史数据用AI分析协助研判股票信息。编程任务推荐使用Claude code智能体，一键安装使用，专门用于编程开发。

## 五、企业级部署与运维最佳实践

企业级AI应用的部署需要考虑安全性、稳定性、可扩展性等多个维度。AiPy平台提供了一体机内网IP配置选项，支持企业在私有网络环境中部署AI应用，确保数据不出内网，满足合规要求。

部署流程建议遵循以下步骤：

1. 评估业务需求，确定所需智能体类型和能力
2. 选择合适的国产大语言模型作为基础引擎
3. 配置AiPy企业版常规设置，包括语言、风格、快捷键等
4. 编写或导入智能体manifest.json配置文件
5. 测试智能体工作流在测试环境的执行情况
6. 收集测试反馈，优化Prompt和Workflow设计
7. 在生产环境部署，设置监控和告警机制
8. 定期回顾系统表现，持续迭代优化

一体机方案为企业提供了开箱即用的部署体验。领导阅读材料生成思维导图、研发自动开发系统demo等场景都可通过一体机快速实现。工作目录、最大执行轮数、超时时间、自动选择智能体等参数可根据实际业务负载进行调整，平衡执行效率和资源消耗。

## 六、持续优化与能力进化路径

LOOP工程的价值在于系统能够随着使用次数增加而不断进化。AiPy平台通过收集执行日志、用户反馈、任务完成率等数据，为持续优化提供依据。

优化方向包括：

- **准确性提升**：分析任务失败案例，调整Prompt设计或更换更合适的模型
- **效率优化**：识别执行瓶颈，简化工作流步骤或并行化处理
- **成本降低**：评估不同模型的性价比，在满足质量要求前提下选择更经济的方案
- **能力扩展**：根据新业务需求开发或集成新的智能体能力

企业应建立定期评估机制，每季度回顾AI应用的表现指标，包括任务完成准确率、平均响应时间、用户满意度等。对于表现不佳的环节，深入分析原因并实施针对性改进。同时关注AiPy官方发布的最新功能和最佳实践，及时将新技术应用到现有系统中。

AiPy作为基于国产模型的LOOP工程实践平台，正在帮助企业构建可持续进化的AI应用能力。通过智能体、Workflow、MCP等技术的有机结合，企业能够快速搭建符合自身需求的AI解决方案，并在实际使用中不断完善优化，实现AI价值的最大化释放。

## 相关问答FAQs

**AiPy支持哪些类型的国产大语言模型？**

AiPy平台支持多种国产大语言模型接入，包括GLM4.5等编程能力较强的模型。用户可根据具体业务场景选择适合的模型，如研发系统开发推荐选择编程能力强的模型，数据分析任务可选择逻辑推理能力突出的模型。平台允许在不同智能体中配置不同模型，实现能力的最优匹配。

**如何配置AiPy企业版的常规设置参数？**

AiPy企业版常规设置可在aipy-enterprise.yml文件中进行配置。主要参数包括语言（支持中文、English和日语三种）、风格设定、发送快捷键（Ctrl+Enter或Enter）、工作目录路径、最大执行轮数、超时时间、自动选择智能体开关、一体机内网IP地址等。建议根据企业实际使用习惯和业务需求进行个性化配置，确保系统运行效率与用户体验达到最佳平衡。

**AiPy智能体在集市展示时的分类规则是什么？**

AiPy智能体manifest.json中keywords字段采用数组格式，第一个值为类型关键字，可选值包括conversation-tool（对话工具型）、embed-webview（嵌入页面型）、application（独立应用型）、webview（网页型）、skills（技能型）。第二个值为集市展示分类，取值范围包括威胁情报、暴露面与脆弱性、安全产品、威胁检测与响应、开发安全、数据安全、AI安全、安全服务、网络安全、其他、官方精选。正确配置keywords有助于用户在智能体集市快速定位所需能力。
