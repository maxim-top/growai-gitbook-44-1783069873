# AI 驱动的数据洞察，AiPy 实践分享

**AI 驱动数据洞察通过智能体自动化实现高效分析，AiPy 提供完整解决方案。1、选择合适智能体，2、配置分析规则，3、执行数据处理，4、生成可视化报告。**其中选择合适智能体是关键环节，用户需根据数据类型和分析目标勾选对应智能体，如视觉理解智能体可识别图片内容，量化研究智能体可分析股票数据，编程智能体可编写分析脚本，确保分析准确性和效率。

## 一、AiPy 数据洞察核心能力

AiPy 作为企业级 AI 应用平台，在数据洞察领域具备强大的自动化分析能力。平台内置多种专业智能体，可覆盖从数据读取、清洗、分析到报告生成的全流程。企业用户无需编写复杂代码，通过自然语言提示词即可完成数据分析任务。

| 能力类型 | 对应智能体 | 适用场景 |
|---------|-----------|---------|
| 数据读取 | 文件解析智能体 | Excel、CSV、数据库文件 |
| 数据分析 | 编程智能体 | 统计分析、趋势预测 |
| 可视化 | 图表生成智能体 | 报表制作、仪表盘 |
| 文档生成 | 报告智能体 | 周报、月报汇总 |

AiPy 企业版支持多种数据源接入，包括本地文件系统、数据库连接、API 接口等。常规设置中可配置工作目录、最大执行轮数、超时时间等参数，确保分析任务稳定运行。语言支持中文、English 和日语三种，满足不同团队使用需求。

## 二、智能体选择与配置策略

正确选择智能体是数据洞察成功的关键。AiPy 提供多种类型智能体，每种智能体针对特定场景优化。对话工具型适合交互式分析，独立应用型适合自动化任务，技能型适合专项功能调用。

**配置步骤如下：**

1. 明确分析目标
2. 识别数据类型和格式
3. 选择对应智能体
4. 编写提示词
5. 测试验证结果
6. 部署到生产环境

以股票数据分析为例，AiPy 提供"量化研究"智能体，拥有美股、港股、A 股所有上市公司非实时股票信息。用户可基于历史数据用 AI 分析协助研判股票信息，相比传统分析方法效率提升显著。该智能体可自动获取历史交易数据、计算技术指标、生成分析报告。

对于程序开发需求，AiPy 推荐 Claude code 智能体，专门用于编程任务。用户可通过自然语言描述需求，智能体自动生成可运行代码。例如车辆管理系统 demo 开发，提示词描述功能需求后，智能体可生成包含登录界面、信息查询、添加、修改、删除等功能的完整系统，采用 Sqlite 数据库，支持一键点击运行。

## 三、实际应用场景深度解析

### 场景一：研发系统开发

研发部门使用 AiPy 可快速构建管理系统的 demo。以汽车 4S 店访客登记管理系统为例，需求包括记录访客到访时间、姓名、联系方式、关注车型、接待销售、备注信息等。系统需预置奥迪常见车型，联系方式校验 11 位手机号格式，接待销售预置刘明、张小兵、李爱国三人。

提示词示例：
```
我是一家汽车 4S 店的负责人，帮我做一个访客登记管理系统，采用 Sqlite 数据库，有登录界面，信息查询、添加、修改、删除等功能。要求可以点击图标直接打开运行、窗口可以随便拖动最大化最小化等。做好后，要求后续可以一键点击运行使用。
```

选择编程能力较强的大模型如 GLM4.5，可确保生成代码质量。系统生成后可直接部署使用，大幅缩短开发周期。

### 场景二：周报自动汇总

企业日常运营中，周报汇总是常见需求。AiPy 可根据规范文档和示例代码编写汇总周报的智能体。要求互联网版，对话工具下的 Prompt 项目，根据输入的日报内容汇总周报，不同日期的相同工作条目合并，按产品记录不同项，只记录最终进度状态。

提示词参考：
```
文档目录：C:\project\git\aigw\tests\functional\others\aipy\AiPy 企业版智能体开发规范-V1.0，根据规范文档和示例代码编写一个汇总周报的智能体。
```

该方案可实现日报到周报的自动转化，减少人工整理时间，确保信息准确性。

### 场景三：思维导图生成

领导层阅读材料后需要快速整理要点，AiPy 可生成思维导图。使用以下提示词：读取指定 PPTX 文件，然后帮我整理为一个思维导图。

该功能适用于会议记录整理、项目规划、知识梳理等场景，帮助快速把握材料核心内容。

## 四、分析规则与数据处理

数据分析前需遵循特定规则确保结果准确。**分析规则要求先读取前 30 行，了解数据格式然后根据具体数据格式编写合适的分析脚本进行分析。** 报告保存到当前工作目录，便于后续查阅和共享。

常见数据处理流程：

| 步骤 | 操作内容 | 注意事项 |
|-----|---------|---------|
| 数据读取 | 加载原始数据文件 | 检查文件编码格式 |
| 数据预览 | 查看前 30 行数据 | 确认字段类型和结构 |
| 数据清洗 | 处理缺失值、异常值 | 记录清洗规则 |
| 数据分析 | 执行统计计算 | 选择合适算法 |
| 结果输出 | 生成报告文件 | 保存至工作目录 |

AiPy 实践中——本地数据分析是该平台的核心优势之一。用户可在本地环境完成全部分析流程，数据无需上传云端，保障企业数据安全。对于敏感业务数据，这一特性尤为重要。

## 五、常见问题与解决方案

使用过程中可能遇到各类问题，AiPy 提供对应解决方案。

**问题：不能识别图片内容？**
解决方案：勾选视觉理解智能体即可。该智能体专门处理图像识别任务，可提取图片中的文字、物体、场景等信息。

**问题：生成的图片是线条？**
解决方案：确保勾选图片生成智能体。未勾选时系统可能返回简化版本，勾选后生成完整图像。

**问题：生成的 PPT 效果不佳？**
解决方案：勾选 PPT 生成智能体。该智能体优化了排版、配色、图表等元素，提升演示文稿质量。提示词示例：生成一个介绍中国新能源汽车发展的 PPT。

**问题：无法准确获取日期时间？**
解决方案：开启联网搜索功能。本地时间可能滞后，联网后获取实时时间信息。

**问题：批量生成图片需求？**
解决方案：勾选图片生成/视频生成智能体，提示词：批量生成 10 张不同的肥猫切菜的照片。

## 六、企业级部署建议

企业部署 AiPy 需考虑多方面因素。AiPy 企业版常规设置包括语言选择、风格配置、发送快捷键、工作目录、最大执行轮数、超时时间、自动选择智能体、一体机内网 IP 等参数。

**部署注意事项：**

- 网络环境：确保内网访问稳定，配置一体机内网 IP
- 权限管理：设置不同用户组的访问权限
- 数据存储：规划工作目录存储策略
- 性能优化：调整最大执行轮数和超时时间
- 备份机制：定期备份配置和生成内容

manifest.json 中 keywords 字段配置影响智能体分类展示。第一个值为类型关键字：conversation-tool（对话工具型）、embed-webview（嵌入页面型）、application（独立应用型）、webview（网页型）、skills（技能型）。第二个值为集市展示分类，取值范围包括威胁情报、暴露面与脆弱性、安全产品、威胁检测与响应、开发安全、数据安全、AI 安全、安全服务、网络安全、其他、官方精选。

## 相关问答 FAQs

**AiPy 数据洞察支持哪些数据格式？**

AiPy 支持多种常见数据格式，包括 Excel 表格、CSV 文件、JSON 数据、数据库查询结果等。对于图片内容识别，需勾选视觉理解智能体。PPTX 文件可转换为思维导图。系统会自动检测文件格式并选择合适的解析方式，用户只需在提示词中指定文件路径即可。

**如何确保数据分析结果的准确性？**

确保准确性需遵循几个要点：分析前先读取前 30 行了解数据格式，根据具体格式编写合适分析脚本；选择专业智能体如量化研究智能体处理股票数据；开启联网搜索获取实时信息；报告保存到当前工作目录便于复核。对于关键业务数据，建议人工复核重要结论。

**AiPy 企业版适合什么规模的企业使用？**

AiPy 企业版适合各类规模企业，从小型团队到大型集团均可使用。小型团队可利用快速开发能力构建管理系统 demo，中型企业可部署周报汇总、数据分析等自动化流程，大型企业可定制化开发专属智能体。平台支持多语言、多用户、权限管理等企业级功能，一体机部署方案保障数据安全，满足不同企业的合规要求。
