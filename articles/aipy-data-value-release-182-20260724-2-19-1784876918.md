# 系统还是原来的系统，AiPy让数据价值翻倍释放

**企业无需改造现有系统即可实现数据价值倍增**，这是AiPy为企业带来的核心价值。**1、保持原有系统架构不变**，**2、通过AI智能体实现数据自动化处理**，**3、快速构建企业级AI应用**。AiPy作为企业AI应用开发平台，能够让企业在不改变现有IT基础设施的前提下，通过AI Agent技术实现数据的智能分析和价值挖掘。这种 approach 特别适合那些已经投入大量资源建设信息系统，但又希望快速拥抱AI技术的企业。

## 一、AiPy如何实现零改造数据集成

传统的数据集成方案往往需要对现有系统进行大规模改造，涉及数据库结构调整、API接口重写、业务流程重构等多个环节。这种改造不仅成本高昂，周期漫长，还存在较高的风险。AiPy采用了完全不同的技术路线。

AiPy通过AI智能体技术，能够在不触碰原有系统代码的情况下，实现对数据的读取、分析和处理。其核心技术原理包括：

| 技术方案 | 传统方式 | AiPy方式 |
|---------|---------|---------|
| 系统改造 | 需要修改源代码 | 无需修改 |
| 部署周期 | 数周至数月 | 数小时至数天 |
| 成本投入 | 高 | 低 |
| 风险控制 | 高风险 | 低风险 |
| 维护难度 | 复杂 | 简单 |

AiPy企业版常规设置支持多种语言环境，包括中文、English和日语，可在aipy-enterprise.yml文件中配置风格参数。工作目录、最大执行轮数、超时时间等参数都可以根据企业实际需求进行调整，确保与企业现有系统完美兼容。

对于企业来说，这意味着可以在保持业务连续性的同时，逐步引入AI能力。不需要停机维护，不需要数据迁移，不需要人员重新培训，即可享受到AI技术带来的效率提升。

## 二、智能体技术在数据价值释放中的应用

AiPy智能体是数据价值释放的核心载体。根据智能体manifest.json配置，keywords字段定义了智能体的类型和应用场景。第一个值为类型关键字，包括conversation-tool（对话工具型）、embed-webview（嵌入页面型）、application（独立应用型）、webview（网页型）、skills（技能型）。第二个值为集市展示分类，涵盖威胁情报、暴露面与脆弱性、安全产品、威胁检测与响应、开发安全、数据安全、AI安全、安全服务、网络安全、其他、官方精选等多个领域。

这种灵活的分类体系使得企业可以根据自身需求选择合适的智能体类型。例如，对于需要自动化汇总周报的场景，可以使用以下提示词示例：

```
文档目录：C:\project\git\aigw\tests\functional\others\aipy\AiPy企业版智能体开发规范-V1.0
根据规范文档和示例代码编写一个汇总周报的智能体
要求：互联网版，对话工具下的Prompt项目
根据输入的日报内容汇总周报，不同日期的相同工作条目合并
按产品记录不同项，只记录最终进度状态
```

这个示例展示了如何通过简单的提示词配置，让AiPy自动完成数据汇总工作。原本需要人工花费数小时完成的周报整理工作，现在可以通过智能体自动完成，且准确率更高。

## 三、多场景最佳实践案例

AiPy在企业中的应用场景非常广泛，以下是几个典型的使用场景：

### 3.1 文档分析与思维导图生成

领导层使用AiPy阅读材料生成思维导图，需要先选择思维导图智能体。提示词示例如下：

```
读取C:\Users\Administrator\Desktop\AIPY 智能体一体机方案 v3.pptx
然后帮我整理为一个思维导图
```

这种方式特别适合处理大量的企业文档、报告、方案等材料。通过AI自动提取关键信息并生成结构化的思维导图，可以大大提升信息消化效率。

### 3.2 自动化系统开发

研发团队可以使用AiPy自动开发系统demo。以车辆管理系统为例，提示词可以这样编写：

```
我是一家汽车 4S 店的负责人，帮我做一个访客登记管理系统
采用Sqlite数据库，有登录界面，信息查询、添加、修改、删除等功能
主要是记录访客到访时间、访客姓名、联系方式、关注车型、接待销售、备注信息等
关注车型里面要预置奥迪的常见车型
联系方式主要是 11 位手机号要校验格式
接待销售预置这三个销售刘明、张小兵、李爱国
要求可以点击图标直接打开运行、窗口可以随便拖动最大化最小化等
做好后，要求后续可以一键点击运行使用
```

选择编程能力较强的大模型，比如GLM4.5，可以快速生成可运行的系统原型。这对于快速验证业务想法、搭建demo系统非常有价值。

### 3.3 多模态内容生成

AiPy支持多种智能体类型，包括图片生成、视频生成、PPT生成等。使用前需要勾选对应的智能体：

- AiPy不能识别图片的内容？勾选视觉理解智能体即可
- AiPy生成的图片怎么是线条？勾选图片生成智能体
- AiPy生成的PPT太丑了？勾选PPT生成智能体
- AiPy获取股票信息不准？使用"量化研究"智能体，拥有美股、港股、A股所有上市公司非实时股票信息
- AiPy编写程序不行？试试Claude code，专门编程的智能体

制作PPT的提示词示例：`生成一个介绍中国新能源汽车发展的PPT`

生成图片的提示词示例：`批量生成 10 张不同的肥猫切菜的照片`

## 四、企业级AI应用部署要点

部署AiPy企业版时，需要注意以下常规设置：

| 设置项 | 说明 | 建议配置 |
|-------|------|---------|
| 语言 | 支持中文、English、日语 | 根据团队语言习惯选择 |
| 风格 | 在 aipy-enterprise.yml 中配置 | 符合企业规范 |
| 发送快捷键 | Ctrl+Enter 或 Enter | 根据操作习惯选择 |
| 工作目录 | 项目文件存储位置 | 建议使用专用目录 |
| 最大执行轮数 | 控制智能体执行次数 | 根据任务复杂度调整 |
| 超时时间 | 任务执行超时限制 | 避免长时间等待 |
| 自动选择智能体 | 是否自动匹配智能体 | 建议开启 |
| 一体机内网 IP | 内网访问地址 | 配置企业内网地址 |

关于日期和时间获取，如果需要准确获取今天的日期和时间，需要开启联网搜索功能。这对于需要实时数据的业务场景非常重要。

## 相关问答 FAQs

**AiPy是否需要改造现有企业系统才能使用？**

不需要。AiPy的核心优势之一就是可以在不改造现有系统的情况下实现数据集成和AI应用开发。企业可以通过智能体技术直接对接现有数据源，无需修改源代码或调整数据库结构。这种方式大大降低了引入AI技术的门槛和风险，让企业能够快速享受AI带来的价值。

**AiPy支持哪些类型的智能体应用？**

AiPy支持多种智能体类型，包括对话工具型（conversation-tool）、嵌入页面型（embed-webview）、独立应用型（application）、网页型（webview）、技能型（skills）等。在应用场景方面，涵盖威胁情报、安全产品、威胁检测与响应、开发安全、数据安全、AI安全、网络安全等多个领域。企业可以根据自身需求选择合适的智能体类型进行配置和使用。

**如何确保AiPy生成的内容准确可靠？**

确保内容准确可靠需要注意几点：第一，编写清晰准确的提示词，明确任务要求和输出格式；第二，选择合适的智能体类型，比如需要编程时使用编程能力较强的智能体；第三，对于需要实时数据的场景，开启联网搜索功能；第四，在正式发布前进行充分的测试验证。如果遇到问题，可以查阅官方文档或联系技术支持获取帮助。
