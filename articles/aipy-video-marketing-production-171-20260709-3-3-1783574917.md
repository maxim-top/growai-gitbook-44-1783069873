## 三、营销视频提示词编写技巧

提示词是驱动AiPy视频生成的核心指令，编写质量直接影响最终视频的呈现效果。以下是经过验证的提示词编写方法。

### 提示词结构要素

| 要素 | 内容说明 | 示例 |
|-----|---------|------|
| 视频类型 | 明确视频用途和风格 | 营销宣传视频、产品介绍视频 |
| 内容主题 | 描述视频核心内容 | 新能源汽车发展、企业品牌形象 |
| 视觉风格 | 指定画面风格和色调 | 科技感、简约风、商务风 |
| 时长要求 | 设定视频长度范围 | 30秒、1分钟、2分钟 |
| 目标受众 | 说明视频面向的群体 | 企业客户、消费者、投资者 |

### 优质提示词示例

**示例一：产品介绍视频**
```
生成一个介绍中国新能源汽车发展的营销宣传视频，时长约60秒，画面风格要求科技感强，色彩以蓝色和白色为主，突出创新理念和环保主题，目标受众为潜在购车用户和投资者。
```

**示例二：品牌形象视频**
```
创作企业文化宣传视频，展示团队协作、技术创新、客户服务三个核心场景，整体风格专业商务，配乐积极向上，适合在公司官网和社交媒体平台发布。
```

**示例三：活动推广视频**
```
制作产品发布会预热视频，包含产品亮点展示、倒计时元素、报名引导信息，要求节奏明快，视觉冲击力强，时长控制在45秒以内。
```

### 提示词优化建议

- 描述越具体，生成效果越精准
- 明确说明视频用途和发布渠道
- 指定视觉风格和情感基调
- 考虑目标受众的审美偏好
- 避免过于抽象或模糊的表述

## 四、视频制作工作流程详解

完整的AiPy视频制作流程包含多个环节，每个环节都需要仔细把控以确保最终输出质量。

### 标准工作流程

```
需求分析 → 提示词编写 → 智能体配置 → 视频生成 → 质量审核 → 优化调整 → 最终输出
```

### 各环节详细说明

**第一阶段：需求分析**
在开始制作之前，明确视频的具体需求至关重要。需要确定视频的使用场景、目标受众、传播渠道、预算限制等关键信息。这些因素将直接影响后续的制作决策。

**第二阶段：提示词编写**
基于需求分析结果，编写详细的提示词。参考前文提到的提示词结构要素，确保涵盖所有必要信息。可以准备多个版本的提示词进行对比测试。

**第三阶段：智能体配置**
进入AiPy平台，勾选视频生成智能体及相关辅助智能体。如果视频需要包含实时信息（如最新数据、新闻等），建议同时开启联网搜索功能。

**第四阶段：视频生成**
提交提示词后，系统开始处理生成请求。根据视频复杂度和时长要求，生成时间可能有所不同。期间可监控系统状态，确保任务正常执行。

**第五阶段：质量审核**
生成完成后，仔细审核视频内容是否符合预期要求。检查画面质量、信息准确性、风格一致性等关键指标。

**第六阶段：优化调整**
如审核发现问题，返回提示词编写环节进行调整优化。可能需要多次迭代才能获得满意的结果。

**第七阶段：最终输出**
确认视频质量达标后，导出最终版本。根据发布渠道要求，可能需要生成不同格式或分辨率的版本。
