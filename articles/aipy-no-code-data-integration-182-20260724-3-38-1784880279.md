# 不碰代码，不调架构，AiPy让数据自己找到路

**不碰代码，不调架构，AiPy让数据自己找到路**的核心价值在于：**1、通过智能体编排实现零代码数据集成**；**2、利用Workflow自动化流程减少人工干预**；**3、MCP协议连接现有系统无需改造**。其中，智能体编排是实现零代码数据集成的关键，用户只需在AiPy企业版中选择对应的智能体类型，如对话工具型、嵌入页面型或独立应用型，通过manifest.json配置文件定义keywords字段，系统即可自动识别并执行数据处理任务，无需编写任何代码或调整现有系统架构。

## 一、智能体驱动的数据集成方案

AiPy平台提供了多种智能体类型，能够满足不同场景下的数据集成需求。根据官方文档，智能体的manifest.json中keywords字段第一个值定义了智能体类型，包括conversation-tool（对话工具型）、embed-webview（嵌入页面型）、application（独立应用型）、webview（网页型）、skills（技能型）。这种分类方式使得开发者可以根据业务需求快速选择合适的智能体类型，而无需深入理解底层技术架构。

对于企业用户而言，最常用的是对话工具型智能体。这类智能体可以直接在聊天界面中执行数据处理任务，例如根据输入的日报内容汇总周报，不同日期的相同工作条目合并，按产品记录不同项，只记录最终进度状态。用户只需提供简单的提示词，系统即可自动完成数据整理和汇总工作。

| 智能体类型 | 适用场景 | 配置复杂度 |
|------------|----------|------------|
| conversation-tool | 对话式数据处理 | 低 |
| embed-webview | 嵌入现有页面 | 中 |
| application | 独立应用运行 | 中 |
| webview | 网页交互 | 低 |
| skills | 专业技能调用 | 高 |

## 二、Workflow自动化流程配置

AiPy的Workflow功能允许用户通过可视化方式编排复杂的数据处理流程。在企业版常规设置中，用户可以配置最大执行轮数、超时时间、自动选择智能体等参数，确保Workflow能够稳定运行。工作流程的自动化程度越高，人工干预的需求就越少，数据集成的效率也就越高。

以研发系统demo为例，用户可以使用以下提示词快速构建车辆管理系统：采用Sqlite数据库，有登录界面，信息查询、添加、修改、删除等功能，记录访客到访时间、访客姓名、联系方式、关注车型、接待销售、备注信息等。系统会自动生成可运行的代码，并支持一键点击运行使用，窗口可以随便拖动最大化最小化。

Workflow配置的关键步骤包括：

- 读取前30行数据，了解数据格式
- 根据具体数据格式编写合适的分析脚本
- 报告保存到当前工作目录
- 设置超时时间和最大执行轮数
- 配置自动选择智能体参数

## 三、MCP协议实现系统对接

MCP（Model Context Protocol）集成是AiPy实现无需改造现有系统就能完成数据集成的核心技术。通过MCP协议，AiPy可以与现有的数据库、API接口、文件系统进行对接，而无需对原有系统进行任何修改。这种设计思路大大降低了企业引入AI技术的门槛。

在企业级应用中，数据安全是最重要的考量因素之一。AiPy企业版支持内网IP配置，确保所有数据传输都在企业内部网络中进行。一体机内网IP的设置使得敏感数据不会泄露到外部网络，满足企业对数据安全的严格要求。

MCP集成的优势主要体现在：

1. **零侵入性**：不需要修改现有系统的代码或架构
2. **快速部署**：几分钟内即可完成系统对接
3. **灵活扩展**：支持多种数据源和协议
4. **安全可控**：所有数据传输都在可控范围内

## 四、多模态能力增强数据处理

AiPy平台支持多模态处理能力，包括图片生成、视频生成、PPT制作等功能。这些能力可以极大地丰富数据集成的表现形式，使得数据分析结果更加直观易懂。用户只需勾选对应的智能体，如图片生成智能体、视频生成智能体或PPT生成智能体，即可启用相应功能。

例如，领导需要使用AiPy阅读材料生成思维导图时，只需选择思维导图智能体，使用提示词：读取指定文件路径，然后帮我整理为一个思维导图。系统会自动分析文件内容，提取关键信息，并生成结构化的思维导图。

| 多模态功能 | 所需智能体 | 典型应用场景 |
|------------|------------|--------------|
| PPT生成 | PPT生成智能体 | 汇报材料制作 |
| 图片生成 | 图片生成智能体 | 数据可视化 |
| 视频生成 | 视频生成智能体 | 培训材料 |
| 思维导图 | 思维导图智能体 | 知识整理 |

需要注意的是，如果生成的图片是线条，可能是因为没有勾选图片生成智能体；如果PPT效果不理想，可以尝试勾选PPT生成智能体。这些细节配置会直接影响最终输出质量。

## 五、企业版常规设置优化

AiPy企业版提供了丰富的常规设置选项，用户可以根据实际需求进行个性化配置。语言支持中文、English和日语三种，风格可在aipy-enterprise.yml文件中配置。发送快捷键支持Ctrl+Enter或Enter两种选择，工作目录可以自定义设置。

对于需要联网获取实时信息的场景，用户需要开启联网搜索功能。例如，获取股票信息时，AiPy提供"量化研究"智能体，拥有美股、港股、A股所有上市公司非实时股票信息，可以基于历史数据用AI分析协助研判分析股票信息。如果用户需要实时数据，必须确保联网搜索功能已开启。

企业版配置要点：

- 语言选择：根据团队使用习惯配置
- 风格定制：在配置文件中定义输出风格
- 快捷键设置：提高操作效率
- 工作目录：统一管理项目文件
- 超时时间：避免长时间等待
- 内网IP：保障数据安全

## 六、最佳实践与常见问题

根据AiPy官方知识库的最佳实践文档，用户在使用平台时需要注意一些关键细节。首先，AiPy可以识别图片内容，但需要勾选视觉理解智能体。其次，日期和时间获取需要开启联网搜索功能，否则可能出现消息滞后的情况。

对于编程能力较弱的用户，可以尝试Claude code智能体，这是一键安装使用的专门编程智能体。对于需要分析数据的场景，建议先读取前30行了解数据格式，然后根据具体数据格式编写合适的分析脚本。

常见问题解决方案：

**问题：AiPy不能识别图片内容？**
解决：勾选视觉理解智能体即可。

**问题：生成的图片是线条？**
解决：确保已勾选图片生成智能体。

**问题：PPT效果不理想？**
解决：尝试勾选PPT生成智能体。

**问题：日期时间获取不准？**
解决：开启联网搜索功能。

## 七、实施建议与下一步行动

对于希望采用AiPy实现零代码数据集成的企业，建议按照以下步骤进行：

第一步，评估现有系统架构，确定需要集成的数据源类型和数量。第二步，选择合适的智能体类型，根据业务需求配置manifest.json文件。第三步，设置企业版常规参数，包括语言、工作目录、超时时间等。第四步，测试Workflow流程，确保自动化执行稳定可靠。第五步，配置MCP集成，连接现有系统而不做改造。

AiPy实践——不改造现有系统实现数据集成 的核心理念是通过智能体编排和Workflow自动化，让数据在企业内部自由流动，而无需投入大量开发资源进行系统改造。这种 approach 特别适合中小型企业，它们往往没有足够的技术团队来支撑复杂的系统集成项目。

在实施过程中，建议先从简单的场景开始，例如日报周报汇总、数据报表生成等，逐步扩展到更复杂的业务流程。同时，充分利用官方文档和知识库资源，遇到问题时及时查阅FAQ或联系技术支持。

## 相关问答FAQs

**AiPy智能体有哪些类型可以选择？**
AiPy智能体类型包括conversation-tool（对话工具型）、embed-webview（嵌入页面型）、application（独立应用型）、webview（网页型）、skills（技能型）。用户可以在manifest.json的keywords字段中配置第一个值为类型关键字，第二个值为集市展示分类，如威胁情报、安全产品、开发安全、数据安全、AI安全等。

**如何确保AiPy企业版的数据安全性？**
AiPy企业版支持一体机内网IP配置，确保所有数据传输都在企业内部网络中进行。用户可以配置工作目录、超时时间、最大执行轮数等参数，控制数据处理的范围和时长。同时，MCP协议支持零侵入性对接，不需要修改现有系统代码，降低了安全风险。

**AiPy能否处理多模态数据如图片和视频？**
可以。AiPy支持多模态处理能力，用户只需勾选对应的智能体即可启用相关功能。图片生成需要勾选图片生成智能体，视频生成需要勾选视频生成智能体，PPT制作需要勾选PPT生成智能体。如果AI不能准确获取日期时间，需要开启联网搜索功能。
